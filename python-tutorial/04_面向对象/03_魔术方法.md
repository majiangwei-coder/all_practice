# 魔术方法

## 什么是魔术方法

以**双下划线 `__` 开头和结尾**的特殊方法，由 Python 解释器调用，实现特定协议。

```python
class MyClass:
    def __init__(self, value):   # 创建实例时调用
        self.value = value

    def __str__(self):            # str() 或 print() 时调用
        return f"MyClass({self.value})"

    def __len__(self):            # len() 时调用
        return self.value

obj = MyClass(42)
print(obj)        # __str__ 被调用
print(len(obj))   # __len__ 被调用
```

## 创建与销毁

```python
class MyClass:
    def __new__(cls, *args, **kwargs):
        """创建实例前调用，返回实例（通常是 super().__new__）"""
        print("创建实例")
        return super().__new__(cls)

    def __init__(self):
        """创建实例后调用，初始化"""
        print("初始化")

    def __del__(self):
        """实例被销毁时调用（垃圾回收）"""
        print("销毁")

obj = MyClass()
# 输出：创建实例
#       初始化
# del obj → 销毁
```

**注意**：`__new__` 几乎不需要重写，只有不可变对象（如 tuple 的子类）才可能需要。

## 字符串表示

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __str__(self):
        """人类可读的字符串（print/str）"""
        return f"{self.name}, {self.age}岁"

    def __repr__(self):
        """开发者可读的字符串（debug/repr）"""
        return f"Person(name={self.name!r}, age={self.age!r})"

p = Person("Alice", 30)
print(p)        # Alice, 30岁
print(repr(p))  # Person(name='Alice', age=30)
```

### `__str__` vs `__repr__`

| 方法 | 用途 | 调用场景 |
|------|------|----------|
| `__str__` | 人类可读 | `print()`, `str()` |
| `__repr__` | 开发者可读 | `repr()`, debug |

## 比较运算符

```python
class Number:
    def __init__(self, value):
        self.value = value

    def __eq__(self, other):
        if isinstance(other, Number):
            return self.value == other.value
        return NotImplemented

    def __lt__(self, other):
        if isinstance(other, Number):
            return self.value < other.value
        return NotImplemented

    def __le__(self, other):
        return self == other or self < other

    def __gt__(self, other):
        return not self <= other

    def __ge__(self, other):
        return not self < other

    def __ne__(self, other):
        return not self == other
```

**技巧**：使用 `functools.total_ordering` 减少代码：

```python
from functools import total_ordering

@total_ordering
class Number:
    def __init__(self, value):
        self.value = value

    def __eq__(self, other):
        return self.value == other.value

    def __lt__(self, other):
        return self.value < other.value

    # 只需实现 __eq__ 和 __lt__，其他比较自动生成
```

## 算术运算符

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        """self + other"""
        return Vector(self.x + other.x, self.y + other.y)

    def __sub__(self, other):
        """self - other"""
        return Vector(self.x - other.x, self.y - other.y)

    def __mul__(self, scalar):
        """self * scalar"""
        return Vector(self.x * scalar, self.y * scalar)

    def __rmul__(self, scalar):
        """scalar * self"""
        return self * scalar

    def __neg__(self):
        """-self"""
        return Vector(-self.x, -self.y)

    def __repr__(self):
        return f"Vector({self.x}, {self.y})"

v1 = Vector(1, 2)
v2 = Vector(3, 4)
print(v1 + v2)      # Vector(4, 6)
print(v1 * 3)       # Vector(3, 6)
print(3 * v1)       # Vector(3, 6) (__rmul__)
print(-v1)          # Vector(-1, -2)
```

## 索引与切片

```python
class MyList:
    def __init__(self, data):
        self.data = list(data)

    def __getitem__(self, index):
        """支持索引和切片"""
        return self.data[index]

    def __setitem__(self, index, value):
        """支持赋值"""
        self.data[index] = value

    def __len__(self):
        return len(self.data)

    def __contains__(self, item):
        """支持 in 操作符"""
        return item in self.data

    def __iter__(self):
        """支持 for 循环"""
        return iter(self.data)

lst = MyList([1, 2, 3, 4, 5])
print(lst[0])       # 1
print(lst[1:3])     # [2, 3]
lst[0] = 10
print(0 in lst)      # False
for x in lst:
    print(x)        # 1 2 3 4 5
```

## callable：像函数一样调用

```python
class Adder:
    def __init__(self, n):
        self.n = n

    def __call__(self, x):
        """使实例可以像函数一样调用"""
        return self.n + x

add5 = Adder(5)
print(add5(10))   # 15 (等价于 add5.__call__(10))
```

## 上下文管理器

```python
class FileManager:
    def __init__(self, filename):
        self.filename = filename

    def __enter__(self):
        """进入 with 块时调用"""
        self.file = open(self.filename, 'r')
        return self.file

    def __exit__(self, exc_type, exc_val, exc_tb):
        """退出 with 块时调用"""
        self.file.close()
        return False  # 不屏蔽异常

with FileManager('test.txt') as f:
    content = f.read()
# 自动关闭文件
```

## 属性访问

```python
class MyClass:
    def __getattr__(self, name):
        """访问不存在的属性时调用"""
        return f"属性 {name} 不存在"

    def __setattr__(self, name, value):
        """设置任意属性时调用"""
        if not isinstance(value, str):
            raise TypeError("必须字符串")
        object.__setattr__(self, name, value)  # 避免递归

    def __delattr__(self, name):
        """删除属性时调用"""
        object.__delattr__(self, name)
```

## 常用魔术方法速查

| 类别 | 方法 | 何时调用 |
|------|------|----------|
| 创建/销毁 | `__new__`, `__init__`, `__del__` | 创建/销毁实例 |
| 字符串 | `__str__`, `__repr__` | `str()`/`print()`, `repr()` |
| 比较 | `__eq__`, `__lt__`, `__le__`... | 比较运算符 |
| 算术 | `__add__`, `__sub__`, `__mul__`... | 算术运算符 |
| 索引 | `__getitem__`, `__setitem__` | `obj[i]` |
| 长度 | `__len__` | `len()` |
| callable | `__call__` | `obj()` |
| 上下文 | `__enter__`, `__exit__` | `with` |
| 迭代 | `__iter__`, `__next__` | `for`/`next()` |
| 属性 | `__getattr__`, `__setattr__` | 访问/设置属性 |

## 小结

1. **魔术方法**是双下划线包围的特殊方法，由 Python 解释器自动调用
2. **`__init__`**：初始化实例；`__new__`**：创建实例（少用）
3. **`__str__` vs `__repr__`**：前者人类可读，后者开发者可读
4. **比较运算符**：使用 `functools.total_ordering` 可减少代码
5. **`__call__`**：让实例可像函数一样被调用
6. **上下文管理器**：`__enter__`/`__exit__` 实现 `with` 协议
