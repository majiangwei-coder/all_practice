# 类与实例

## 类的定义

```python
class 类名:
    """文档字符串"""
    类属性 = 值      # 类属性（所有实例共享）

    def __init__(self, 参数):
        self.实例属性 = 参数

    def 实例方法(self):
        return "方法"
```

## `__init__`：构造函数

```python
class Dog:
    def __init__(self, name, age):
        self.name = name      # 实例属性
        self.age = age

    def bark(self):
        return f"{self.name} says woof!"

# 创建实例
my_dog = Dog("Buddy", 3)
print(my_dog.name)    # "Buddy"
print(my_dog.bark())  # "Buddy says woof!"
```

## self 是什么

`self` 是**实例对象本身**，类似其他语言的 `this`。

```python
class A:
    def __init__(self, x):
        self.x = x     # self.x 是实例属性

    def show(self):
        print(self.x)  # self 指向当前实例
```

## 类属性 vs 实例属性

```python
class Dog:
    species = "Canis familiaris"  # 类属性：所有实例共享

    def __init__(self, name):
        self.name = name          # 实例属性：每个实例独立
```

### 图示

```
类属性 vs 实例属性：

类 (Dog)
  ├── species = "Canis..."  ← 类属性，存储在类中
  │
实例 (my_dog)        实例 (your_dog)
  ├── name = "Buddy"    ├── name = "Max"
  └── (引用 Dog)        └── (引用 Dog)
```

### 访问优先级

```python
my_dog = Dog("Buddy")
your_dog = Dog("Max")

my_dog.species         # "Canis familiaris" (继承自类)
my_dog.name            # "Buddy" (自己的)

# 修改类属性
Dog.species = "Felis catus"
print(my_dog.species)  # "Felis catus" (所有实例都受影响)

# 修改实例属性，不影响其他实例
my_dog.name = "Rex"
print(my_dog.name)     # "Rex"
print(your_dog.name)   # "Max" (不变)
```

## 实例方法、类方法、静态方法

```python
class MyClass:
    class_attr = "类属性"

    def __init__(self, value):
        self.value = value       # 实例属性

    def instance_method(self):
        """只能访问实例属性和类属性"""
        return f"实例方法: {self.value}"

    @classmethod
    def class_method(cls):
        """可以访问类属性，不能访问实例属性"""
        return f"类方法: {cls.class_attr}"

    @staticmethod
    def static_method():
        """不能访问实例和类属性（类似普通函数）"""
        return "静态方法"
```

### 调用方式

```python
obj = MyClass(42)

obj.instance_method()       # 通过实例调用
MyClass.instance_method(obj)  # 通过类调用（需传 self）

MyClass.class_method()      # 通过类调用
obj.class_method()         # 通过实例调用

MyClass.static_method()    # 通过类调用
obj.static_method()        # 通过实例调用
```

## 数据封装：私有属性

Python 没有真正的访问控制，通过命名约定：

```python
class BankAccount:
    def __init__(self, balance):
        self._balance = balance    # "受保护"，约定不直接访问
        self.__pin = "1234"        # "私有"，会被 name mangling

    def get_balance(self):
        return self._balance

# 访问受保护属性（不推荐但可以）
print(account._balance)

# 访问私有属性（会被改名）
print(account._BankAccount__pin)   # "1234"
```

### 命名约定

| 约定 | 含义 |
|------|------|
| `_protected` | 约定不直接外部访问 |
| `__private` | 名称会被改写（name mangling） |

## 特殊属性

```python
class MyClass:
    """文档字符串"""

obj = MyClass()

obj.__class__           # <class '__main__.MyClass'>
MyClass.__name__       # 'MyClass'
MyClass.__doc__         # '文档字符串'
obj.__dict__            # {'属性字典': ...}
```

## 类与实例的关系

```python
class Dog:
    def __init__(self, name):
        self.name = name

# 创建实例时发生的事：
# 1. 创建新对象
# 2. 调用 __init__(self, name)
# 3. self 指向新对象，设置属性
# 4. 返回新对象

# 类是模板，实例是根据模板创建的具体对象
```

## 小结

1. **`__init__`**：构造函数，创建实例时自动调用
2. **self**：指向实例本身，类似其他语言的 this
3. **类属性 vs 实例属性**：类属性共享，实例属性独立
4. **实例方法 vs 类方法 vs 静态方法**：
   - 实例方法：`self` 作首参
   - 类方法：`cls` 作首参，用 `@classmethod`
   - 静态方法：无隐式参数，用 `@staticmethod`
5. **私有属性**：`_protected` 和 `__private` 是约定，不是强制
