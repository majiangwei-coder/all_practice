# Python 从入门到进阶 — 完整教程

> 面向零基础到中级开发者，从语法入门到工程实战。每个概念从"是什么"开始，深入"为什么"和"怎么用"。

## 目标读者

- 编程零基础，想系统学习 Python
- 有其他语言经验，想深入掌握 Python 特性
- 希望不只是"会用"，更要理解背后的原理和最佳实践

## 学习路线

```
第一章: 语法基础
        → 变量与数据类型、运算符与表达式、控制流、关键字详解、输入输出

第二章: 数据结构
        → list/tuple/dict/set/str 核心机制与性能原理
        → 其他重要类型 (bytes, deque, defaultdict, Counter, namedtuple, typing)

第三章: 函数
        → 函数定义与调用、参数详解、lambda与闭包、装饰器

第四章: 面向对象
        → 类与实例、继承与多态、魔术方法、属性管理(@property/描述符/__slots__)

第五章: 异常与调试
        → 异常处理、调试技术

第六章: 进阶主题
        → 生成器与迭代器、上下文管理器、模块与包、类型提示详解

第七章: 数据处理
        → 文件与IO、序列化与JSON、正则表达式

第八章: 并发编程
        → 多线程、多进程、asyncio 异步

第九章: 标准库精选
        → collections、itertools与functools、datetime与time、pathlib

第十章: 工程实践
        → 虚拟环境与包管理、测试基础、日志系统、项目结构与配置

附录:
        → Python禅与编码规范、常用内置函数速查
```

## 深入程度

本教程不止步于"怎么用"，更关注"为什么"：

- 数据结构的时间/空间复杂度与底层实现
- 函数的参数传递机制、LEGB 作用域
- 类与对象的内存模型、属性访问协议
- GIL 与并发模型的取舍
- 异步编程的协程模型
- 工程化的最佳实践（测试、日志、项目结构）
- 标准库的设计思想与高级用法

## 如何使用

1. 按章节顺序阅读 Markdown 文档
2. 手打代码，不要复制粘贴
3. 完成每章末尾的思考题
4. 修改示例代码，验证自己的理解
5. 学完前六章后可跳到第十章了解工程实践
