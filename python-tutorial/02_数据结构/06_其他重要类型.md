# 其他重要类型

除了 list/tuple/dict/set/str，Python 还有这些重要的内置类型。

## bytes 与 bytearray

### bytes：不可变字节序列

```python
# 创建
b = b"hello"           # 字面量
b = bytes([104, 101, 108, 108, 111])  # 从列表
b = "hello".encode()  # 字符串编码

# 索引返回 int
b[0]    # 104

# 切片返回 bytes
b[1:3]  # b'el'

# 不能修改
b[0] = 100  # TypeError!
```

### bytearray：可变字节序列

```python
ba = bytearray(b"hello")
ba[0] = 100  # 允许修改
ba.append(33)
print(ba)    # bytearray(b'dello!')
```

### 用途：网络编程、文件二进制数据

```python
# 处理 HTTP 请求
request = b"GET / HTTP/1.1\r\nHost: example.com\r\n\r\n"

# 读取二进制文件
with open("image.png", "rb") as f:
    data = f.read()  # bytes
```

## array.array：高效数值数组

```python
from array import array

# typecode 指定数据类型
arr = array('i', [1, 2, 3, 4, 5])  # 'i' = signed int
arr = array('f', [1.0, 2.0, 3.0])  # 'f' = float
arr = array('d', [1.0, 2.0, 3.0])  # 'd' = double

# 比 list 更省内存（特别是数值类型）
import sys
list_arr = [1] * 1000
array_arr = array('i', [1] * 1000)
sys.getsizeof(list_arr)    # ~8000 bytes
sys.getsizeof(array_arr)   # ~4000 bytes
```

### typecode 参考

| typecode | C 类型 | Python 类型 | 大小 |
|----------|--------|------------|------|
| 'b' | signed char | int | 1 |
| 'B' | unsigned char | int | 1 |
| 'h' | short | int | 2 |
| 'i' | int | int | 4 |
| 'l' | long | int | 4/8 |
| 'f' | float | float | 4 |
| 'd' | double | float | 8 |

## collections.deque：双端队列

```python
from collections import deque

dq = deque([1, 2, 3])

# 两端操作 O(1)
dq.append(4)       # 右端添加
dq.appendleft(0)    # 左端添加
dq.pop()           # 右端弹出
dq.popleft()        # 左端弹出

# 常用场景
dq = deque(maxlen=5)  # 固定长度，超出自动丢弃旧元素
for i in range(10):
    dq.append(i)
print(dq)  # deque([5, 6, 7, 8, 9], maxlen=5)
```

## collections.defaultdict

```python
from collections import defaultdict

# 访问不存在的 key 自动创建默认值
dd = defaultdict(list)
dd["fruits"].append("apple")
dd["fruits"].append("banana")
# {'fruits': ['apple', 'banana']}

# 设置默认工厂
dd = defaultdict(int)   # 默认 0
dd["a"] += 1  # {"a": 1}

dd = defaultdict(set)   # 默认空 set
dd["colors"].add("red")
```

## collections.OrderedDict

```python
from collections import OrderedDict

# Python 3.7+ 普通 dict 已保证顺序
# OrderedDict 在早期版本有用，现在主要为了兼容性

od = OrderedDict()
od["z"] = 1
od["a"] = 2
od["m"] = 3
list(od.keys())  # ['z', 'a', 'm']
```

## collections.Counter

```python
from collections import Counter

# 统计元素出现次数
cnt = Counter(["a", "b", "a", "c", "a", "b"])
# Counter({'a': 3, 'b': 2, 'c': 1})

# 常用方法
cnt.most_common(2)     # [('a', 3), ('b', 2)]
cnt.total()           # 6 (Python 3.10+)
list(cnt.elements())  # ['a', 'a', 'a', 'b', 'b', 'c']

# 更新计数
cnt.update(["a", "d"])
cnt["a"] += 1
```

## collections.namedtuple

```python
from collections import namedtuple

# 创建命名字段元组
Point = namedtuple('Point', ['x', 'y'])
p = Point(3, 5)
print(p.x, p.y)       # 3, 5
print(p[0], p[1])     # 3, 5 (可用索引)

# 自动生成方法
p._asdict()           # {'x': 3, 'y': 5}
p._fields             # ('x', 'y')
Point._make([7, 9])   # Point(x=7, y=9)

# 替换某些字段
p2 = p._replace(x=10)  # Point(x=10, y=5)
```

## collections ChainMap

```python
from collections import ChainMap

# 合并多个字典（视图，不复制）
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}
cm = ChainMap(dict1, dict2)

print(cm["a"])  # 1 (第一个字典)
print(cm["b"])  # 2 (第一个字典的 b，优先级高)
print(cm["c"])  # 4 (第二个字典)

# 用途：模拟作用域链
local_vars = {"x": 1}
global_vars = {"x": 10, "y": 20}
scope = ChainMap(local_vars, global_vars)
print(scope["x"])  # 1 (local 优先)
```

## typing 模块：类型提示

```python
from typing import List, Dict, Tuple, Set, Optional, Union, Callable

# 变量类型提示
x: int = 5
name: str = "Alice"

# 函数类型提示
def greet(name: str) -> str:
    return f"Hello, {name}"

# 复杂类型
def process(items: List[int]) -> Dict[str, int]:
    return {"count": len(items)}

# Optional：可能为 None
def find_user(user_id: int) -> Optional[str]:
    if user_id > 0:
        return "found"
    return None

# Union：多种类型
def parse(value: Union[int, str]) -> float:
    return float(value)

# Callable：函数类型
def apply(func: Callable[[int, int], int], a: int, b: int) -> int:
    return func(a, b)
```

### typing 常用类型

| 类型 | 说明 |
|------|------|
| `List[int]` | int 列表 |
| `Dict[str, int]` | str 到 int 的字典 |
| `Tuple[int, str]` | 元组，固定两个元素 |
| `Set[int]` | int 集合 |
| `Optional[int]` | int 或 None |
| `Union[int, str]` | int 或 str |
| `Callable[[int], str]` | 接收 int 返回 str 的函数 |
| `Any` | 任意类型 |
| `NoReturn` | 从不返回（如 sys.exit） |

## range 对象

```python
# range(start, stop, step)
r = range(5)        # 0, 1, 2, 3, 4
r = range(2, 8)     # 2, 3, 4, 5, 6, 7
r = range(0, 10, 2) # 0, 2, 4, 6, 8

# 切片（返回新 range）
r[2:4]     # range(4, 8)

# 转换
list(range(5))      # [0, 1, 2, 3, 4]
tuple(range(5))     # (0, 1, 2, 3, 4)

# 成员测试 O(1)
1000000 in range(1000000000)  # True (不遍历)
```

## memoryview

```python
# 共享内存视图，避免复制
data = bytearray(b"hello")
mv = memoryview(data)
print(mv[0])        # 104
print(mv[1:3].tobytes())  # b'el'

# 修改原始数据
mv[0] = 72  # 'H'
print(data)  # bytearray(b'Hello')
```

## frozenset：不可变集合

```python
# 不可变，可哈希，可作为 dict key 或 set 元素
fs = frozenset([1, 2, 3])

# 可以嵌套
s = {frozenset([1, 2]), frozenset([3, 4])}

# 用途：作为字典 key
inventory = {
    frozenset(["apple", "fruit"]): 10
}
```

## 小结

1. **bytes/bytearray**：二进制数据处理
2. **array**：高效数值数组，比 list 省内存
3. **deque**：双端队列，高效的头尾操作
4. **defaultdict**：访问不存在的 key 自动创建默认值
5. **Counter**：计数统计
6. **namedtuple**：带名字的轻量级结构
7. **typing**：类型提示，提高代码可读性
8. **range**：惰性序列，O(1) 成员测试
9. **frozenset**：不可变集合，可哈希
