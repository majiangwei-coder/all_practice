# 关键字详解

Python 有 35 个关键字，不能用作标识符。理解它们有助于深入理解语言设计。

## 关键字一览

```
False, None, True, and, as, assert, async, await,
break, class, continue, def, del, elif, else, except,
finally, for, from, global, if, import, in, is,
lambda, nonlocal, not, or, pass, raise, return,
try, while, with, yield
```

## 真假值相关：True / False / None

### True / False

布尔值，是 int 的子类：

```python
True == 1    # True
False == 0   # True
isinstance(True, int)  # True

# 运算时 True=1, False=0
print(True + True + False)  # 2
```

### None

表示"无"或"空"，单例对象：

```python
x = None
x is None    # True（判断 None 的标准方式）

def func():
    return None  # 无返回值时隐式返回

# None 的 bool 值
bool(None)   # False
```

## 逻辑运算：and / or / not

### and

```python
# 短路：左边为假则返回左边，否则返回右边
0 and 2     # 0
1 and 2     # 2
"a" and "b" # "b"
```

### or

```python
# 短路：左边为真则返回左边，否则返回右边
0 or 2     # 2
1 or 2     # 1
"" or "default"  # "default"
```

### not

```python
# 始终返回 bool
not 0      # True
not ""     # True
not None   # True
not "hi"   # False
```

### 优先级：not > and > or

```python
# not True or False → (not True) or False → False or False → False
not True or False   # False

# not (True or False) → not True → False
not (True or False) # False

# 实际建议：加括号
```

## 函数定义：def / return / lambda / yield

### def

```python
def greet(name):
    return f"Hello, {name}"
```

### return

```python
def func():
    return          # 返回 None
    # return None   # 等价

def func2():
    return 1, 2, 3  # 返回 tuple

# 条件返回
def find(items, target):
    for i, item in enumerate(items):
        if item == target:
            return i
    return -1
```

### lambda

```python
square = lambda x: x ** 2
add = lambda a, b: a + b
```

### yield

```python
def gen():
    yield 1
    yield 2

g = gen()
next(g)  # 1
next(g)  # 2
```

## 异常处理：try / except / else / finally / raise / assert

### try / except / else / finally

```python
try:
    # 可能出错的代码
    x = 1 / 0
except ZeroDivisionError:
    # 处理特定异常
    x = 0
except (ValueError, TypeError):
    # 处理多种异常
    x = None
except Exception as e:
    # 捕获所有其他异常
    raise  # 重新抛出
else:
    # try 中无异常时执行
    print("成功")
finally:
    # 始终执行
    print("清理")
```

### raise

```python
raise ValueError("invalid value")
raise  # 重新抛出当前异常

# 异常链
raise RuntimeError("failed") from e
raise RuntimeError("failed") from None  # 隐藏原始异常
```

### assert 深入

```python
# assert condition, message
# 条件为 False 时抛出 AssertionError

# 断言函数参数
def calculate_percentage(value, total):
    assert total != 0, "总数不能为零"
    assert 0 <= value <= total, f"值 {value} 不在合理范围 [0, {total}]"
    return value / total * 100

# assert 不是错误处理！
# 以下用法是错的：
# try:
#     x = int(input("输入: "))
#     assert x > 0
# except AssertionError:
#     ...

# assert 用于"不应该发生"的检查（程序员错误，不是用户错误）
# assert 可以通过 -O 参数禁用：python -O script.py
# 正式的错误处理请用 if...raise

# assert 作为"可执行注释"
def process_data(data, mode):
    assert mode in ("fast", "accurate"), f"mode 必须是 fast 或 accurate, 不是 {mode}"
    # ...
```

### del 深入

```python
# del 删除的是引用（标签），不是对象

# 1. 删除变量
x = 42
del x
# print(x)  # NameError

# 2. 删除列表元素（按索引）
lst = [1, 2, 3, 4, 5]
del lst[0]      # [2, 3, 4, 5]
del lst[1:3]    # [2, 5]

# 3. 删除字典键
d = {"a": 1, "b": 2, "c": 3}
del d["b"]      # {"a": 1, "c": 3}

# 4. 删除属性
class MyClass:
    x = 1

obj = MyClass()
obj.y = 2
del obj.y
# del MyClass.x  # 删除类属性

# 好的做法：del 后对象引用计数减1，可能触发垃圾回收
# 不要手动调 gc，Python 自己会处理
```

### type 深入

```python
# type() 的两个完全不同的用途

# 用途1：获取对象的类型
print(type(42))       # <class 'int'>
print(type("hello"))  # <class 'str'>
print(type([1,2,3]))  # <class 'list'>

# 用途2：动态创建类（元类编程）
# type(name, bases, dict) → 创建新类

# 等同于 class Dog: pass
Dog = type('Dog', (), {})

# 等同于 class Animal:
#           species = "unknown"
#           def speak(self):
#               return "..."
Animal = type('Animal', (), {
    'species': 'unknown',
    'speak': lambda self: '...'
})

# 带继承
# class Cat(Animal): pass
Cat = type('Cat', (Animal,), {})

print(Cat().speak())  # ...
print(Cat.species)    # unknown

# 实际项目中很少直接使用 type() 创建类
# 通常用 class 关键字，或用元类 (metaclass)
```

### import / from 深入

```python
# import 的多种形态

# 1. 导入模块
import os
import os.path          # 子模块
import numpy as np      # 别名

# 2. 从模块导入
from os import path     # 导入单个
from os import path, listdir  # 导入多个
from os import path as p      # 导入并改名

# 3. 相对导入（包内使用）
from . import sibling   # 同包的兄弟模块
from .sibling import func
from .. import parent_module
from ..package import other

# 4. 条件导入
try:
    import ujson as json  # 更快
except ImportError:
    import json           # 回退到标准库

# 5. 延迟导入（解决循环导入）
def my_func():
    import heavy_module  # 只在需要时导入
    return heavy_module.process()
```

## 其他容易被忽略的关键字

### `with` 的多种用法

```python
# 不只用于文件
import threading

# 锁
lock = threading.Lock()
with lock:
    # 自动 acquire/release
    pass

# 抑制异常
from contextlib import suppress
with suppress(FileNotFoundError):
    os.remove("nonexistent.txt")

# 重定向输出
from contextlib import redirect_stdout
import io

f = io.StringIO()
with redirect_stdout(f):
    print("这行不会被显示")
print(f.getvalue())  # "这行不会被显示\n"
```

### `in` 的多种用法

```python
# 1. 成员测试
3 in [1, 2, 3]        # True
'key' in {'key': 1}   # True（检查 key）

# 2. for 循环
for i in range(5): ...

# 3. 字典和生成器推导式
{x: y for x, y in pairs}

# 4. 字符串搜索（比 .find() 更可读）
if 'error' in log_line:
    print("日志中有错误")
```

### `as` 的多种用法

```python
# 1. import 别名
import numpy as np

# 2. with 绑定
with open('f.txt') as file:
    ...

# 3. except 绑定
except ValueError as e:
    ...

# 4. match 绑定（Python 3.10+）
match point:
    case (x, y) as p:
        print(f"点 {p} 的坐标是 ({x}, {y})")
```

### `pass` vs `...` (Ellipsis)

```python
# pass：什么都不做（占位符）
def todo():
    pass  # 以后再实现

# ... (Ellipsis)：也是合法的占位符
def todo():
    ...  # 功能相同

# 区别：
# pass 更有"这里是空的"的语义
# ... 常用于类型存根（stub files）和协议

# 在 match 中使用
match x:
    case 1:
        ...
    case 2:
        print("two")
```

## 作用域：global / nonlocal

### global

```python
x = "global"

def func():
    global x  # 声明使用全局 x
    x = "modified"

func()
print(x)  # "modified"
```

### nonlocal

```python
def outer():
    x = "outer"

    def inner():
        nonlocal x  # 声明使用外层函数作用域的 x
        x = "inner"

    inner()
    print(x)  # "inner"

outer()
```

## 其他：pass / del / type

### pass

```python
# 空代码块占位
if condition:
    pass
else:
    do_something()

def empty_func():
    pass  # 函数体不能为空

class EmptyClass:
    pass
```

### del

```python
x = 1
del x  # 删除变量，对象可能被垃圾回收

d = {"a": 1, "b": 2}
del d["a"]  # 删除字典元素

# del 删除的是引用，不是对象
# 如果对象没有其他引用，才会被垃圾回收
```

### type

```python
# type() 函数
type(42)       # <class 'int'>
type("str")   # <class 'str'>

# type 作为类
class MyClass:
    pass

# type 动态创建类（不推荐，类装饰器更好）
Dog = type('Dog', (object,), {'bark': lambda self: 'woof'})
```

## 关键字与内置函数的区别

```python
# 这些是关键字，不能覆盖
# if, else, for, while, def, class, import, etc.

# 这些是内置函数，可以覆盖但不推荐
# len, print, str, int, list, dict, etc.

# 覆盖内置函数的危险
list = [1, 2, 3]  # 覆盖内置 list
# list([1, 2, 3])  # 报错！

# 恢复
del list  # 或 import builtins; list = builtins.list
```

## 小结

1. **35个关键字**：保留字，不能用作标识符
2. **逻辑**：and/or/not，注意短路求值
3. **作用域**：global 修改全局，nonlocal 修改外层
4. **异常**：try/except/finally/raise/assert
5. **异步**：async/await 配合使用
6. **不要覆盖内置函数**：如 list, str, dict 等
