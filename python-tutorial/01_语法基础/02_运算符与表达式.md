# 运算符与表达式

## 算术运算符

| 运算符 | 说明 | 示例 | 注意 |
|--------|------|------|------|
| `+` | 加 | `3 + 2` → `5` | 也用于拼接字符串 |
| `-` | 减 | `3 - 2` → `1` | |
| `*` | 乘 | `3 * 2` → `6` | 也用于重复字符串 |
| `/` | 除 | `3 / 2` → `1.5` | **总是 float** |
| `//` | 整除 | `3 // 2` → `1` | 向下取整 |
| `%` | 取余 | `3 % 2` → `1` | |
| `**` | 幂 | `3 ** 2` → `9` | `2 ** 3` = 8 |

### 整除的坑

```python
print(-3 // 2)   # -2 (向下取整，不是截断)
print(3 // -2)   # -2
print(-3 % 2)    # 1  (Python 保证 a % b 与 b 同号)
```

## 比较运算符

返回 `bool` 值：

```python
print(3 > 2)     # True
print(3 >= 3)    # True
print(3 < 3)     # False
print(3 <= 3)    # True
print(3 == 3)    # True
print(3 != 2)    # True
```

### 链式比较

Python 支持**链式比较**，非常优雅：

```python
# 等价于 1 < 3 and 3 < 5
print(1 < 3 < 5)           # True

# 等价于 1 < 3 and 3 < 5 and 5 < 10
print(1 < 3 < 5 < 10)      # True

# 判断范围
x = 5
print(1 < x < 10)          # True

# 常见错误：不理解链式比较
# 下面这种写法在 Python 中是错误的（不会按预期工作）：
# if x == 1 or 2 or 3:  # 永远为 True，因为 2, 3 是 truthy
```

## 逻辑运算符

| 运算符 | 说明 | 特点 |
|--------|------|------|
| `and` | 逻辑与 | 短路求值 |
| `or` | 逻辑或 | 短路求值 |
| `not` | 逻辑非 | |

### 短路求值

```python
# and：左边为 False 则直接返回左边，不计算右边
print(0 and 2)     # 0 (返回左边)
print(1 and 2)     # 2 (返回右边)

# or：左边为 True 则直接返回左边，不计算右边
print(1 or 2)      # 1 (返回左边)
print(0 or 2)      # 2 (返回右边)
```

### 实战用法

```python
# 设置默认值
name = input_name or "匿名"

# 短路防止除零
x = 0
if x != 0 and 1 / x > 0.5:
    print("ok")

# 简化条件
if is_admin and has_permission:
    print("通过")
```

## 赋值运算符

```python
# 基本赋值
x = 1

# 增量赋值
x += 1   # x = x + 1
x -= 1   # x = x - 1
x *= 2   # x = x * 2
x /= 2   # x = x / 2
x //= 2  # x = x // 2
x %= 3   # x = x % 3
x **= 2  # x = x ** 2
```

## 位运算符

| 运算符 | 说明 |
|--------|------|
| `&` | 按位与 |
| `|` | 按位或 |
| `^` | 按位异或 |
| `~` | 按位取反 |
| `<<` | 左移 |
| `>>` | 右移 |

```python
# 常用技巧
# 判断奇偶
n = 5
print(n & 1)    # 1 (奇数)
print(n & 1 == 0) # False (奇数)

n = 4
print(n & 1)    # 0 (偶数)
print(n & 1 == 0) # True (偶数)

# 快速乘除 2
print(4 << 1)    # 8  (乘2)
print(8 >> 1)   # 4  (除2)
```

## 成员运算符

```python
# in / not in
print(3 in [1, 2, 3])       # True
print("a" in "abc")         # True
print("x" not in "abc")     # True

# 字典：检查 key
d = {"a": 1, "b": 2}
print("a" in d)             # True
print(1 in d)               # False (检查的是 key，不是 value)
```

## 身份运算符

```python
# is / is not（已在前章介绍）
a = None
if a is None:
    print("是 None")
```

## 运算符优先级

从高到低（牢记前几条即可）：

1. `**` 幂运算
2. `*`, `/`, `//`, `%` 乘除
3. `+`, `-` 加减
4. `>`, `<`, `>=`, `<=` 比较
5. `==`, `!=` 相等
6. `=` 赋值
7. `is`, `is not`
8. `in`, `not in`
9. `not`
10. `and`
11. `or`

### 建议：加括号

记不清优先级时就加括号，代码更清晰：

```python
# 这样更明确
if (a > 0) and (b > 0):
    print("都是正数")
```

## 三元表达式

```python
# 语法：value_if_true if condition else value_if_false
age = 20
status = "成年" if age >= 18 else "未成年"
print(status)  # "成年"

# 可以嵌套（但不推荐，可读性差）
x = 5
result = "正" if x > 0 else ("零" if x == 0 else "负")

# 实际场景
# 打印时根据条件选择内容
name = "Alice"
print(f"你好，{name if name else '访客'}")

# 列表推导式中使用
numbers = [1, -2, 3, -4, 5]
abs_values = [n if n >= 0 else -n for n in numbers]
print(abs_values)  # [1, 2, 3, 4, 5]
```

## 海象运算符 := (Python 3.8+)

`:=` 可以在**表达式内部**进行赋值，避免重复计算：

```python
# 之前：需要两行
data = input("输入: ")
while data != "quit":
    print(f"你输入了: {data}")
    data = input("输入: ")

# 海象运算符：一行搞定
while (data := input("输入: ")) != "quit":
    print(f"你输入了: {data}")

# 另一个常见场景：避免重复计算
# 不用海象
import re
text = "price: $99.99"
match = re.search(r'\d+\.\d+', text)
if match:
    print(float(match.group()))  # match.group() 被调用一次

# 用海象
if match := re.search(r'\d+\.\d+', text):
    print(float(match.group()))

# 列表推导中使用
# 只保留满足条件的计算结果
def expensive(x):
    return x ** 2

# 不用海象：expensive 被调用两次
result = [expensive(x) for x in range(10) if expensive(x) > 10]

# 用海象：只计算一次
result = [y for x in range(10) if (y := expensive(x)) > 10]
```

## 表达式 vs 语句

- **表达式**：有返回值，如 `3 + 2`、`x if x > 0 else -x`
- **语句**：执行操作，没有返回值，如 `if`、`for`、`赋值`

```python
# 赋值语句，没有返回值
x = 3

# 下面这行会报错：SyntaxError
# print(x = 3)

# 但表达式可以
print(x := 3)   # 3 (海象运算符，Python 3.8+)
```

## 小结

1. `/` 除法总是返回 float，要整数除法用 `//`
2. 链式比较 `1 < x < 10` 是 Python 的优雅语法
3. `and`/`or` 有短路求值特性，可用于简化代码
4. 位运算在特定场景下很高效（奇偶判断、快速乘除）
5. 运算符优先级不确定时，加括号最安全
