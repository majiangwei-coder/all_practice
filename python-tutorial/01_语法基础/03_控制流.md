# 控制流

## 条件语句：if / elif / else

### 基本语法

```python
if 条件1:
    代码块
elif 条件2:
    代码块
else:
    代码块
```

### 关键点

- **条件**：必须是 `bool` 值，非 bool 会自动转换
- **缩进**：Python 以缩进划分代码块（4 空格）
- **elif**：相当于 `else if`

### 隐式 bool 转换

以下值在条件中视为 `False`（**Falsy**）：

| 值 | 说明 |
|----|------|
| `None` | 空 |
| `False` | 假 |
| `0`, `0.0` | 零 |
| `""`, `''` | 空字符串 |
| `[]`, `()`, `{}` | 空容器 |
| `set()` | 空集合 |

**其余都是 True（Truthy）**

```python
if []:          # False，不执行
    print("不会打印")

if [0]:         # True，执行！([0] 非空)
    print("会打印")
```

### 避免常见错误

```python
# 错误：赋值作为条件（= 不是 ==）
if x = 10:      # SyntaxError!

# 正确：比较
if x == 10:
    print("x 是 10")

# 错误：多个条件没显式连接
if x > 0 and x < 10  # 正确
```

## 循环语句：for / while

### for 循环

```python
# 基本语法
for 变量 in 可迭代对象:
    代码块

# 示例
for i in [1, 2, 3]:
    print(i)
```

### while 循环

```python
while 条件:
    代码块
```

### break / continue / else

```python
# break：跳出循环
for i in range(10):
    if i == 5:
        break
    print(i)   # 0 1 2 3 4

# continue：跳过本次迭代
for i in range(5):
    if i == 2:
        continue
    print(i)   # 0 1 3 4

# else：循环正常结束（未 break）时执行
for i in range(5):
    pass
else:
    print("循环正常结束")   # 会打印

# break 时不执行 else
for i in range(5):
    if i == 3:
        break
else:
    print("不会打印")
```

### 循环的 else 用途

`else` 在循环中的作用常被忽视，用于**搜索失败**的场景：

```python
# 查找质数
for n in range(2, 10):
    for x in range(2, n):
        if n % x == 0:
            print(f"{n} 不是质数")
            break
    else:
        print(f"{n} 是质数")
```

## range 对象

`range()` 生成**惰性序列**，不占内存：

```python
range(5)        # 0, 1, 2, 3, 4
range(2, 5)     # 2, 3, 4
range(0, 10, 2) # 0, 2, 4, 6, 8 (步长2)
range(5, 0, -1) # 5, 4, 3, 2, 1 (倒序)

# 注意：range 不包含结束值
```

### 高效遍历

```python
# 遍历 0~99
for i in range(100):
    pass

# 遍历列表（获取索引和值）
items = ["a", "b", "c"]
for i, v in enumerate(items):
    print(i, v)
# 0 a
# 1 b
# 2 c

# 遍历多个序列
a = [1, 2, 3]
b = ["x", "y", "z"]
for i, j in zip(a, b):
    print(i, j)
```

## 列表推导式

```python
# 基本：[表达式 for 变量 in 可迭代对象]
[x * 2 for x in range(5)]        # [0, 2, 4, 6, 8]

# 带条件：[表达式 for 变量 in 可迭代对象 if 条件]
[x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# 多层循环
[(x, y) for x in [1, 2] for y in ["a", "b"]]
# [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]
```

### 推导式 vs 普通循环

```python
# 普通循环
result = []
for x in range(10):
    if x % 2 == 0:
        result.append(x * 2)

# 等价的列表推导式
result = [x * 2 for x in range(10) if x % 2 == 0]
```

### 谨慎使用的情况

```python
# 过于复杂的推导式应避免
# 建议：拆成普通循环，保持可读性

# 不推荐：
result = [func(x) for x in items if x > 0 and check(func(x)) and another_check(x)]

# 推荐：
filtered = [x for x in items if x > 0]
transformed = [func(x) for x in filtered if check(x)]
```

## 字典/集合推导式

```python
# 字典推导式
d = {x: x ** 2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# 集合推导式
s = {x for x in range(5)}
# {0, 1, 2, 3, 4}

# 反向字典（交换 k/v）
original = {"a": 1, "b": 2}
reversed_d = {v: k for k, v in original.items()}
# {1: 'a', 2: 'b'}
```

## match 语句（Python 3.10+）

类似其他语言的 `switch`，但更强大：

```python
def http_status(status):
    match status:
        case 200:
            return "OK"
        case 404:
            return "Not Found"
        case 500:
            return "Server Error"
        case _ if status >= 400 and status < 500:
            return "Client Error"
        case _:
            return "Unknown"
```

### 解构匹配（最强大的特性）

```python
# 匹配元组
point = (3, 5)

match point:
    case (0, 0):
        print("原点")
    case (x, 0):
        print(f"在x轴上，x={x}")
    case (0, y):
        print(f"在y轴上，y={y}")
    case (x, y):
        print(f"点({x}, {y})")
    case _:
        print("未知")

# 匹配列表（匹配第一个元素）
def process_command(cmd):
    match cmd:
        case ["quit"]:
            return "退出"
        case ["go", direction]:
            return f"走向{direction}"
        case ["pick", item, count]:
            return f"拾取{count}个{item}"
        case ["pick", *items]:
            return f"拾取{items}"
        case _:
            return "未知命令"

print(process_command(["go", "north"]))    # 走向north
print(process_command(["pick", "苹果", 3])) # 拾取3个苹果
print(process_command(["pick", "苹果", "香蕉", "橘子"]))  # 拾取['苹果', '香蕉', '橘子']

# 匹配字典（用类更优雅）
def process_event(event):
    match event:
        case {"type": "click", "x": x, "y": y}:
            return f"点击位置({x}, {y})"
        case {"type": "keypress", "key": key}:
            return f"按下按键{key}"
        case {"type": type_name}:
            return f"未知事件类型: {type_name}"
```

### match 配合 class

```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

def classify_point(p):
    match p:
        case Point(x=0, y=0):
            return "原点"
        case Point(x=0, y=y):
            return f"Y轴, y={y}"
        case Point(x=x, y=0):
            return f"X轴, x={x}"
        case Point(x=x, y=y) if x == y:
            return f"在线 y=x 上"
        case Point():
            return "普通点"
```

### 使用 OR 模式

```python
match status:
    case 401 | 403 | 407:
        return "认证相关错误"
    case 404 | 410:
        return "资源不存在"
    case 500 | 502 | 503:
        return "服务器错误"
```

## 常见模式

### 无限循环 + break

```python
while True:
    user_input = input("输入 'quit' 退出: ")
    if user_input == "quit":
        break
```

### 迭代直到条件满足

```python
# 不断尝试直到成功
while True:
    try:
        value = int(input("输入一个正数: "))
        if value > 0:
            break
    except ValueError:
        pass
```

## 小结

1. **if**：非 bool 值会自动转 bool，注意 `==` 和 `=` 的区别
2. **for**：遍历可迭代对象，`range()` 不占额外内存
3. **while**：注意避免死循环
4. **break/continue/else**：`else` 在循环正常结束时执行
5. **推导式**：简洁但不过度复杂化
6. **match**：Python 3.10+ 的新语法，比 switch 更强大
