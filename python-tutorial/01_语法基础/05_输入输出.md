# 输入与输出

> 程序与用户交互的桥梁：print() 输出和 input() 输入。本章深入讲解格式化输出、f-string、以及命令行参数。

---

## 1. print() 输出

### 基本用法

```python
# 最简单的输出
print("Hello, World!")  # Hello, World!

# 输出多个值（默认用空格分隔）
print("Python", "版本", 3.12)  # Python 版本 3.12

# 自定义分隔符
print("a", "b", "c", sep="-")  # a-b-c
print("2024", "03", "15", sep="/")  # 2024/03/15

# 自定义结尾（默认换行 \n）
print("Hello", end="")
print("World")  # HelloWorld（连在一起）

print("加载中", end="...")
print("完成")  # 加载中...完成

# 输出到文件
with open("output.txt", "w") as f:
    print("写入文件的内容", file=f)
```

### print 的完整签名

```python
print(*objects, sep=' ', end='\n', file=None, flush=False)

# sep: 多个值之间的分隔符
# end: 末尾附加的字符串（默认换行）
# file: 输出目标（默认 sys.stdout）
# flush: 是否立即刷新缓冲区
```

### 进度条效果

```python
import time

# 模拟进度条
for i in range(101):
    print(f"\r进度: [{'=' * (i // 2)}{' ' * (50 - i // 2)}] {i}%", end="", flush=True)
    time.sleep(0.02)
print()  # 换行
# 输出：进度: [==================================================] 100%
```

## 2. input() 输入

### 基本用法

```python
# 读取一行用户输入（返回字符串）
name = input("请输入你的名字: ")
print(f"你好, {name}!")

# input 的返回值永远是字符串！
age = input("请输入年龄: ")
print(type(age))  # <class 'str'>
# 需要数字时必须转换
age = int(input("请输入年龄: "))
```

### 输入处理模式

```python
# 模式1：读取并转换
number = int(input("请输入一个整数: "))

# 模式2：读取并分割多个值
x, y = input("请输入两个数（空格分隔）: ").split()
x = float(x)
y = float(y)
print(f"{x} + {y} = {x + y}")

# 模式3：一次读取多个整数
numbers = list(map(int, input("请输入多个数（空格分隔）: ").split()))
print(f"总和: {sum(numbers)}")

# 模式4：读取多行（直到空行）
print("请输入多行文本（空行结束）:")
lines = []
while True:
    line = input()
    if line == "":
        break
    lines.append(line)
print(f"共输入 {len(lines)} 行")
```

### 输入的安全处理

```python
# 安全的整数输入（带重试）
def get_int(prompt):
    while True:
        try:
            return int(input(prompt))
        except ValueError:
            print("输入无效！请输入一个整数。")

age = get_int("请输入年龄: ")
print(f"年龄: {age}")

# 安全的选项输入（限制范围）
def get_choice(prompt, choices):
    while True:
        choice = input(prompt)
        if choice in choices:
            return choice
        print(f"请输入 {choices} 中的一个")

color = get_choice("选择颜色 (red/green/blue): ", ["red", "green", "blue"])
print(f"你选择了: {color}")
```

## 3. f-string 格式化

Python 3.6+ 的 f-string 是字符串格式化的最佳方式：

```python
name = "Alice"
age = 30
score = 95.6789

# 基本插值
f"我是 {name}，今年 {age} 岁"  # '我是 Alice，今年 30 岁'

# 表达式
f"明年 {age + 1} 岁"  # '明年 31 岁'
f"3 + 5 = {3 + 5}"    # '3 + 5 = 8'

# 调用函数
f"大写: {name.upper()}"  # '大写: ALICE'

# 数字格式化
f"分数: {score:.2f}"       # '分数: 95.68'（保留2位小数）
f"百分比: {0.8567:.1%}"    # '百分比: 85.7%'
f"金额: {1234567:,}元"     # '金额: 1,234,567元'
f"二进制: {42:b}"          # '二进制: 101010'
f"十六进制: {255:#x}"      # '十六进制: 0xff'

# 对齐与填充
f"|{name:<10}|"   # '|Alice     |' 左对齐，宽度10
f"|{name:>10}|"   # '|     Alice|' 右对齐
f"|{name:^10}|"   # '|  Alice   |' 居中
f"|{name:*^10}|"  # '|**Alice***|' 星号填充居中

# 日期时间
from datetime import datetime
now = datetime.now()
f"当前时间: {now:%Y-%m-%d %H:%M:%S}"  # '当前时间: 2024-03-15 10:30:45'

# 调试模式（Python 3.8+）
f"{name=}, {age=}"  # "name='Alice', age=30"

# 多行 f-string
f"""
姓名: {name}
年龄: {age}
分数: {score:.1f}
"""
```

### f-string 速查表

| 格式化 | 语法 | 示例 | 输出 |
|--------|------|------|------|
| 小数位数 | `{x:.2f}` | `f"{3.14159:.2f}"` | `'3.14'` |
| 百分比 | `{x:.1%}` | `f"{0.85:.1%}"` | `'85.0%'` |
| 千位分隔 | `{x:,}` | `f"{1234567:,}"` | `'1,234,567'` |
| 左对齐 | `{x:<10}` | `f"{'hi':<10}"` | `'hi        '` |
| 右对齐 | `{x:>10}` | `f"{'hi':>10}"` | `'        hi'` |
| 居中 | `{x:^10}` | `f"{'hi':^10}"` | `'    hi    '` |
| 补零 | `{x:0>5}` | `f"{42:0>5}"` | `'00042'` |
| 二进制 | `{x:b}` | `f"{5:b}"` | `'101'` |
| 八进制 | `{x:o}` | `f"{8:o}"` | `'10'` |
| 十六进制 | `{x:x}` | `f"{255:x}"` | `'ff'` |

## 4. format() 方法

f-string 出现之前的旧方式，仍需了解：

```python
# 位置参数
"{} + {} = {}".format(1, 2, 3)     # '1 + 2 = 3'
"{1} + {0} = {2}".format(1, 2, 3) # '2 + 1 = 3'（指定顺序）

# 关键字参数
"{name} is {age}".format(name="Alice", age=30)

# 混用
"{} vs {other}".format("this", other="that")

# 解包字典
data = {"name": "Bob", "age": 25}
"{name}, {age}".format(**data)  # 'Bob, 25'

# 对齐和格式化（与 f-string 类似）
"{:<10}".format("left")    # 'left      '
"{:>10}".format("right")   # '     right'
"{:.2f}".format(3.14159)   # '3.14'
```

## 5. % 格式化（最古老）

旧式字符串格式化，了解即可：

```python
"%s is %d years old" % ("Alice", 30)  # 'Alice is 30 years old'
"pi = %.2f" % 3.14159                  # 'pi = 3.14'
"name: %(name)s, age: %(age)d" % {"name": "Bob", "age": 25}
```

| 格式符 | 类型 |
|--------|------|
| `%s` | 字符串 |
| `%d` | 整数 |
| `%f` | 浮点数 |
| `%.2f` | 浮点数（2位小数） |
| `%x` / `%o` | 十六进制 / 八进制 |
| `%%` | 字面百分号 |

**建议**：新代码用 f-string，只有维护旧代码时才用 format() 或 %。

## 6. sys.stdout 与 sys.stderr

```python
import sys

# 标准输出
sys.stdout.write("Hello\n")  # 比 print 底层

# 标准错误（错误信息）
sys.stderr.write("错误：文件未找到\n")

# 重定向输出
import io
old_stdout = sys.stdout
sys.stdout = io.StringIO()
print("这条信息被捕获了")
captured = sys.stdout.getvalue()
sys.stdout = old_stdout
print(f"捕获到的内容: {captured}")
```

## 7. 命令行参数

### sys.argv

```python
import sys

# sys.argv[0] 是脚本名，后续是参数
print(f"脚本名: {sys.argv[0]}")
print(f"参数: {sys.argv[1:]}")

# 使用示例
if len(sys.argv) > 1:
    name = sys.argv[1]
    print(f"Hello, {name}!")
else:
    print("用法: python script.py <name>")
```

### argparse（更强大的命令行解析）

```python
import argparse

parser = argparse.ArgumentParser(description="示例程序")
parser.add_argument("name", help="你的名字")
parser.add_argument("-a", "--age", type=int, default=0, help="年龄")
parser.add_argument("-v", "--verbose", action="store_true", help="详细输出")

args = parser.parse_args()

if args.verbose:
    print(f"处理 {args.name} 的请求...")

print(f"你好, {args.name}!")
if args.age:
    print(f"你 {args.age} 岁")
```

运行示例：
```bash
python script.py Alice --age 30 -v
# 处理 Alice 的请求...
# 你好, Alice!
# 你 30 岁
```

## 8. 实战练习

### 练习1：简易计算器

```python
print("=== 简易计算器 ===")
a = float(input("第一个数: "))
op = input("运算符 (+, -, *, /): ")
b = float(input("第二个数: "))

match op:
    case "+":
        result = a + b
    case "-":
        result = a - b
    case "*":
        result = a * b
    case "/":
        if b != 0:
            result = a / b
        else:
            result = "错误：除以零"
    case _:
        result = "未知运算符"

print(f"{a} {op} {b} = {result}")
```

## 9. 小结

1. **print()**：输出到控制台，控制分隔符（sep）和结尾（end）
2. **input()**：读取用户输入，永远返回字符串，记得类型转换
3. **f-string**：现代格式化首选，功能最全
4. **format()**：旧代码常见，了解即可
5. **% 格式化**：最古老，新代码应避免
6. **sys.argv / argparse**：处理命令行参数
7. **sys.stderr**：输出错误信息
8. **flush=True**：实时输出（如进度条）

### 常见错误

```python
# 错误1：忘记 input 返回字符串
age = input("年龄: ")
# print(age + 1)  # TypeError!
age = int(input("年龄: "))  # 正确

# 错误2：f-string 中引号不匹配
# f"他说："你好""    # SyntaxError

# 错误3：print 调试代码忘记删除
```
