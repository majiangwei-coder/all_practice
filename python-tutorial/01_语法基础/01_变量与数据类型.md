# 变量与数据类型

> 本章讲解 Python 的核心概念：一切皆对象。理解这个概念，你就能理解 Python 的很多行为。

---

## 1. 一切皆对象

### 什么是对象？

在 Python 中，**所有数据都是对象**。这包括数字、字符串、列表，甚至函数也是对象。

每个对象都有三个特征：

| 特征 | 说明 | 比喻 |
|------|------|------|
| **id（身份）** | 对象的内存地址，唯一标识 | 人的身份证号 |
| **type（类型）** | 决定对象能做什么 | 物种类型（人/狗/猫） |
| **value（值）** | 对象存储的数据 | 人的名字 |

### 代码示例

```python
# 创建一个整数 42
a = 42

# id() 函数返回对象的内存地址（不同电脑可能不同）
print("a 的 id:", id(a))
# 输出类似：140...（一串数字，代表内存地址）

# type() 返回对象的类型
print("a 的类型:", type(a))
# 输出：<class 'int'>  （class 是"类"的意思，int 是整数类）

# value 就是对象的值
print("a 的值:", a)
# 输出：42
```

## 2. 内存模型（堆与栈）

理解 Python 的内存管理，能解释很多"奇怪"的行为。

### 图解：栈与堆

```
栈 (Stack)  —  变量名（引用）
    a ──────────→
    b ──────────→

堆 (Heap)   —  对象（数据）
    int(42)  ←────┘
    list([1,2,3]) ←┘
```

- **栈**：存变量名（引用），大小固定，速度快
- **堆**：存对象本身，大小动态，速度较慢

```python
# a 和 id(a) 的关系
a = [1, 2, 3]
# 栈上的 "a" → 存储了一个地址 0x7f... 
# 堆上的 0x7f...  → 存储了列表对象 [1, 2, 3]
# id(a) 返回的就是这个地址
```

### 引用计数与垃圾回收

```python
import sys

a = [1, 2, 3]
print(sys.getrefcount(a))  # 2（a 引用了一次，getrefcount 参数也引用了一次）

b = a  # 增加引用
print(sys.getrefcount(a))  # 3

del b  # 减少引用
# 当引用计数归零 → 对象被垃圾回收 → 内存释放
```

**总结**: Python 自动管理内存。变量只是标签，对象是实际数据。没标签贴上时，对象被回收。

## 3. 核心数据类型一览

Python 有很多内置数据类型，先记住最常用的 8 个：

```python
# ========== 不可变类型（创建后不能修改）==========

整数   (int)     : 42, -7, 0, 999
浮点数 (float)   : 3.14, -0.5, 0.0
布尔值 (bool)    : True, False
字符串 (str)    : "hello", 'world', """多行"""
元组   (tuple)   : (1, 2, 3), ("a", "b")

# ========== 可变类型（创建后可以修改）==========

列表   (list)    : [1, 2, 3], ["a", "b", "c"]
字典   (dict)    : {"name": "Alice", "age": 30}
集合   (set)     : {1, 2, 3}, {"apple", "banana"}
```

### 什么是"可变"与"不可变"？

这是 Python 非常重要的概念：

```python
# ========== 不可变类型示例 ==========

# 字符串是不可变的，不能修改其中的字符
s = "hello"
# s[0] = "H"   # 这行代码会报错！TypeError: 'str' object does not support item assignment

# 正确的做法是创建新字符串
s = "H" + s[1:]  # "Hello" （s[1:] 取从位置1到结尾的子串）
print(s)  # Hello


# ========== 可变类型示例 ==========

# 列表是可变的，可以随时修改
lst = [1, 2, 3]
lst[0] = 100       # 直接修改第一个元素
lst.append(4)       # 末尾添加新元素
lst.insert(1, 50)  # 在位置1插入50
print(lst)  # [100, 50, 2, 3, 4]
```

## 3. 变量本质：标签而非容器

### 颠覆认知的概念

在其他一些语言中（如 C、Java），变量像是"盒子"，把值放进去。
**但在 Python 中，变量是"标签"，贴在对象上。**

```python
# 创建列表 [1, 2, 3]，a 是贴在上面的标签
a = [1, 2, 3]

# b 也贴在这个对象上，不是复制！
b = a

print("a:", a)  # [1, 2, 3]
print("b:", b)  # [1, 2, 3]

# 通过 b 修改这个对象
b.append(4)

print("修改后 a:", a)  # [1, 2, 3, 4]
print("修改后 b:", b)  # [1, 2, 3, 4]
# 注意！a 和 b 都变了，因为它们指向同一个对象！
```

### 图解

```
    a ──────┐
            ├──→ [1, 2, 3, 4]  ← 这是同一个对象
    b ──────┘
```

### 如果想复制（创建独立对象）怎么办？

```python
# 方法1：切片复制（创建新列表）
a = [1, 2, 3]
b = a[:]      # 切片复制，b 是新的独立列表
b.append(4)
print("a:", a)  # [1, 2, 3]  ← a 不变！
print("b:", b)  # [1, 2, 3, 4]

# 方法2：copy() 方法
c = a.copy()

# 方法3：list() 构造函数
d = list(a)

# 方法4：import copy 后用 deepcopy（后面章节详讲）
```

## 4. is 与 == 的区别

这是 Python 新手常犯的错误，一定要理解清楚！

| 运算符 | 作用 | 比喻 |
|--------|------|------|
| `==` | 比较**值**是否相等 | 双胞胎，长相一样（值相同） |
| `is` | 比较**身份**（是否是同一个对象） | 同一个人（身份相同） |

```python
# == 比较值
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)  # True  ← 值相同，所以相等

# is 比较是否是同一个对象
print(a is b)  # False ← 虽然值相同，但是两个不同的对象

# a 和 b 指向不同的内存地址
print(id(a))   # 某个地址，如 140...
print(id(b))   # 另一个地址，如 141...

# 只有指向同一个对象时，is 才返回 True
c = a
print(a is c)   # True  ← c 和 a 指向同一个对象
```

### 什么时候用 is？

`is` 主要用于判断是否是 `None`：

```python
# 判断 None 的标准方式（必须用 is）
x = None
if x is None:
    print("x 是空值")

# 不要用 == 判断 None
# if x == None:  # 可以，但不是最佳实践
```

## 5. 整数缓存机制（了解即可）

Python 为了优化性能，对**小整数**做了缓存：

```python
# 整数缓存范围：-5 到 256
a = 256
b = 256
print(a is b)   # True  ← 在缓存范围内，是同一个对象

a = 257
b = 257
print(a is b)   # False ← 超过缓存范围，是不同对象

# 注意：这是 Python 的实现细节，不要依赖这个行为！
# 判断整数相等永远用 ==，不要用 is
print(a == b)  # True  ← 这个才是正确的比较方式
```

## 6. None 的特殊身份

`None` 表示"空"或"无"，它是一个特殊对象：

```python
# None 是单例：全局只有一个 None 对象
a = None
b = None
print(a is b)   # True  ← 始终是同一个对象

# 判断 None 的正确方式
if a is None:
    print("a 是 None")

# None 的布尔值是 False
print(bool(None))  # False

# None 的典型用途：表示"尚未设置"
name = None  # 尚未有名字
name = "Alice"  # 后来赋值
```

## 7. 类型转换

### 显式转换（手动转换）

```python
# 字符串 → 整数
num = int("42")      # "42" → 42
print(num, type(num))  # 42 <class 'int'>

# 字符串 → 浮点数
pi = float("3.14")   # "3.14" → 3.14
print(pi, type(pi))   # 3.14 <class 'float'>

# 整数 → 字符串
s = str(42)           # 42 → "42"
print(s, type(s))     # 42 <class 'str'>

# 整数 → 浮点数（自动升级）
f = float(42)        # 42 → 42.0
print(f, type(f))     # 42.0 <class 'float'>
```

### 布尔转换（真假判断）

```python
# 转为布尔值
print(bool(0))        # False  ← 零是假
print(bool(1))        # True   ← 非零是真

print(bool(""))       # False  ← 空字符串是假
print(bool(" "))      # True   ← 空格字符串是真
print(bool("hello"))  # True   ← 有内容是真

print(bool([]))       # False  ← 空列表是假
print(bool([1]))      # True   ← 有元素是真

print(bool(None))     # False  ← None 是假
```

### 隐式转换（自动发生）

Python 会自动进行某些类型转换：

```python
# int + float → float（整数被"升级"为浮点数）
result = 3 + 3.0     # 3 + 3.0 → 6.0
print(result, type(result))  # 6.0 <class 'float'>

# bool + int → int（True=1, False=0）
result = True + 1    # True(1) + 1 → 2
print(result)         # 2

result = False + 1    # False(0) + 1 → 1
print(result)         # 1
```

## 9. 浅拷贝与深拷贝

前面简单提到过复制，这里深入讲解。

### 浅拷贝（只复制外层）

```python
import copy

# 原始列表的嵌套结构
original = [[1, 2], [3, 4]]

# 浅拷贝：3种等价方式
shallow1 = original[:]
shallow2 = original.copy()
shallow3 = list(original)
shallow4 = copy.copy(original)

# 修改外层的元素 → 不影响原对象
shallow1.append([5, 6])
print(original)   # [[1, 2], [3, 4]] ← 不变（外层是独立的）
print(shallow1)   # [[1, 2], [3, 4], [5, 6]]

# 但！修改内层元素 → 影哬原对象！
shallow1[0].append(99)
print(original)   # [[1, 2, 99], [3, 4]] ← 内层变了！
print(shallow1)   # [[1, 2, 99], [3, 4], [5, 6]]
```

### 深拷贝（完全复制）

```python
deep = copy.deepcopy(original)  # 递归复制所有层级

deep[0].append(100)
print(original)  # [[1, 2, 99], [3, 4]] ← 不变
print(deep)      # [[1, 2, 99, 100], [3, 4]] ← 完全独立
```

### 图解

```
浅拷贝：
  original ──→ [[1,2], [3,4]]
                 ↑      ↑
  shallow  ─────┘      │
                 └──────┘ (内层列表是共享的!，不是复制)

深拷贝：
  original ──→ [[1,2], [3,4]]

  deep     ──→ [[1,2], [3,4]]  ← 全部独立复制
```

### 日常建议

- 只有一层（扁平列表）→ `.copy()` 或 `[:]` 就够了
- 有嵌套结构 → 用 `copy.deepcopy()`
- 自定义对象 → 可能需要实现 `__deepcopy__` 方法

## 10. 类型转换进阶

### 隐式转换（自动发生）

Python 是**动态类型**语言：

- 变量可以随时指向不同类型的对象
- 但对象本身有固定类型，不能混用

```python
# 变量 x 一开始指向整数
x = 42
print(type(x))  # <class 'int'>

# 后来 x 指向字符串
x = "hello"
print(type(x))  # <class 'str'>

# 这是允许的，因为 x 是变量，可以指向任何对象

# 但对象本身不能跨类型操作
# "hello" + 42  # 报错！TypeError
```

## 9. 小结与常见误区

### 必须记住

1. **一切皆对象**：变量是引用（标签），不是容器
2. **可变 vs 不可变**：可变对象可以被直接修改，不可变对象只能替换
3. **`is` vs `==`**：`is` 比较身份（是否同一个对象），`==` 比较值
4. **`is None` 是判断 None 的标准方式**
5. **小整数缓存是实现细节，永远用 `==` 比较整数**

### 新手常见错误

```python
# 错误1：以为 a = [1,2,3] 后再 b = a 会复制
a = [1, 2, 3]
b = a          # b 和 a 指向同一个对象！
b.append(4)   # 修改 b，a 也会变

# 正确做法：
b = a[:]      # 切片复制
b = a.copy()  # copy 方法

# 错误2：用 is 比较值
a = 1000
b = 1000
# a is b  # False！因为大整数不在缓存范围
a == b        # True ← 正确方式
```
