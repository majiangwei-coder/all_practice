# 生成器与迭代器

## 迭代器协议

迭代器是实现了以下方法的对象：

```python
# __iter__：返回迭代器本身
# __next__：返回下一个元素，无元素时抛出 StopIteration

my_iter = iter([1, 2, 3])
print(next(my_iter))  # 1
print(next(my_iter))  # 2
print(next(my_iter))  # 3
print(next(my_iter))  # StopIteration
```

## 可迭代对象 vs 迭代器

| 类型 | 说明 | 示例 |
|------|------|------|
| 可迭代对象 | 能生成迭代器的对象 | list, str, dict, set |
| 迭代器 | 本身是迭代器对象 | iter([1,2,3]) |

```python
# list 是可迭代对象，不是迭代器
lst = [1, 2, 3]
lst_iter = iter(lst)  # iter() 创建迭代器

print(iterable is iterator)  # False (通常)
print(next(iterable))  # TypeError: 'list' object is not an iterator
```

## 生成器函数

**生成器函数**：包含 `yield` 的函数，调用时返回生成器对象。

```python
def count_up_to(n):
    """生成 1 到 n 的生成器"""
    i = 1
    while i <= n:
        yield i      # 暂停，返回值
        i += 1

# 调用函数返回生成器对象
generator = count_up_to(3)
print(type(generator))  # <class 'generator'>

# 迭代获取值
for num in count_up_to(3):
    print(num)  # 1, 2, 3
```

### 图示：生成器执行流程

```

    i = 1
    while i <= 3:
        yield i      ← 第一次 next() 返回 1，暂停
        i += 1

next(gen) → 1

next(gen) → 3

```

## 生成器 vs 普通函数

| 特性 | 普通函数 | 生成器函数 |
|------|---------|-----------|
| return | 结束函数，返回值 | - |
| yield | - | 暂停函数，返回值 |
| 调用结果 | 返回值 | 生成器对象 |
| 内存 | 一次性全部返回 | 按需生成，惰性 |

```python
# 普通函数：一次返回整个列表
def list_numbers(n):
    result = []
    for i in range(n):
        result.append(i)
    return result

# 生成器函数：按需生成
def gen_numbers(n):
    for i in range(n):
        yield i
```

### 内存对比

```python
# 如果 n = 10亿
# list: 需要 10亿 * 8字节 = 8GB 内存
# generator: 只需几个字节
```

## 生成器表达式

类似列表推导式，但用圆括号，**惰性求值**：

```python
# 列表推导式（立即求值）
list_comp = [x ** 2 for x in range(1000000)]  # 立即占用大内存

# 生成器表达式（惰性求值）
gen_exp = (x ** 2 for x in range(1000000))  # 不立即计算

# 需要时再取值
print(next(gen_exp))  # 0
print(next(gen_exp))  # 1
```

## itertools 模块

```python
import itertools

# count: 无限计数器
counter = itertools.count(start=1, step=2)  # 1, 3, 5, 7...
print(next(counter))  # 1
print(next(counter))  # 3

# cycle: 无限循环
cycler = itertools.cycle([1, 2, 3])  # 1, 2, 3, 1, 2, 3...
print(next(cycler))  # 1
print(next(cycler))  # 2

# repeat: 重复
rep = itertools.repeat("hello", times=3)  # hello, hello, hello

# chain: 连接多个迭代器
chained = itertools.chain([1, 2], ['a', 'b'])
print(list(chained))  # [1, 2, 'a', 'b']

# islice: 切片
sliced = itertools.islice(range(10), 2, 8, 2)  # 2, 4, 6
print(list(sliced))  # [2, 4, 6]

# compress: 按掩码筛选
data = ['a', 'b', 'c', 'd']
mask = [1, 0, 1, 0]
print(list(itertools.compress(data, mask)))  # ['a', 'c']
```

## 实际应用

### 1. 逐行读取大文件

```python
def read_large_file(filename):
    """逐行读取，不会把整个文件加载到内存"""
    with open(filename, 'r') as f:
        for line in f:
            yield line.strip()

for line in read_large_file("huge_log.txt"):
    process(line)
```

### 2. 管道处理

```python
def filter_positive(numbers):
    for n in numbers:
        if n > 0:
            yield n

def square(numbers):
    for n in numbers:
        yield n ** 2

def take(n, iterable):
    for i, item in enumerate(iterable):
        if i >= n:
            break
        yield item

# 惰性管道，不提前计算
pipeline = square(filter_positive([-2, -1, 0, 1, 2, 3]))
print(take(3, pipeline))  # [1, 4, 9]
```

### 3. 斐波那契（无限序列）

```python
def fibonacci():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fibs = fibonacci()
print(next(fibs))  # 0
print(next(fibs))  # 1
print(next(fibs))  # 1
print(next(fibs))  # 2
```

## send() 与 yield from

```python
# 生成器可以用 send() 发送值
def counter():
    count = 0
    while True:
        increment = yield count
        if increment is not None:
            count += increment
        else:
            count += 1

c = counter()
print(next(c))    # 0 (启动生成器)
print(c.send(5))  # 5 (发送 5，count += 5)
print(next(c))    # 6 (increment 为 None，count += 1)
```

### yield from：委托生成器

```python
def gen1():
    yield 1
    yield 2

def gen2():
    yield from gen1()  # 委托给 gen1
    yield 3

print(list(gen2()))  # [1, 2, 3]
```

## 小结

1. **迭代器协议**：`__iter__` + `__next__`
2. **生成器函数**：包含 `yield`，返回生成器对象
3. **惰性求值**：生成器按需生成，节省内存
4. **生成器表达式**：`gen = (x for x in items)`
5. **itertools**：常用迭代工具，count/cycle/chain/islice 等
6. **应用场景**：大文件处理、无限序列、数据管道

## 实战练习

### 练习1：用生成器实现滑动窗口

```python
def sliding_window(iterable, size):
    """生成滑动窗口，惰性处理"""
    from collections import deque
    window = deque(maxlen=size)
    for item in iterable:
        window.append(item)
        if len(window) == size:
            yield tuple(window)

# 移动平均
prices = [10, 12, 11, 13, 14, 15, 13, 12]
for w in sliding_window(prices, 3):
    print(f"{w} → 均值: {sum(w)/3:.1f}")
# (10, 12, 11) → 均值: 11.0
# (12, 11, 13) → 均值: 12.0
# ...
```

### 练习2：生成器委托（yield from 实战）

```python
def read_all_files(file_list):
    """逐行读取多个文件，像一个连续的流"""
    for filename in file_list:
        with open(filename) as f:
            yield from f  # 委托给文件对象（它本身就是迭代器）

# 使用
# for line in read_all_files(["a.txt", "b.txt", "c.txt"]):
#     process(line)
```

### 练习3：双向通信（send 实战）

```python
def running_average():
    """接收数值，维护移动平均值"""
    total = 0
    count = 0
    average = 0
    while True:
        x = yield average  # 接收新值，返回当前均值
        total += x
        count += 1
        average = total / count

avg = running_average()
next(avg)         # 启动生成器到第一个 yield
print(avg.send(10))  # 10.0
print(avg.send(20))  # 15.0
print(avg.send(30))  # 20.0
print(avg.send(40))  # 25.0
```
