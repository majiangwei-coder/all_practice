# 参数详解

## 参数类型一览

| 类型 | 语法 | 说明 |
|------|------|------|
| 位置参数 | `def f(a, b)` | 必须按顺序传递 |
| 默认参数 | `def f(a=1)` | 可选，有默认值 |
| 可变位置参数 | `def f(*args)` | 收集成 tuple |
| 可变关键字参数 | `def f(**kwargs)` | 收集成 dict |
| 关键字-only 参数 | `def f(*, a, b)` | 必须用关键字传参 |
| 位置-or-关键字 | `def f(a, *, b)` | a 位置，b 关键字 |

## 位置参数

```python
def power(base, exponent):
    return base ** exponent

power(2, 3)   # 8
power(3, 2)   # 9
```

## 默认参数

```python
def power(base, exponent=2):
    return base ** exponent

power(2)      # 4 (exponent=2)
power(2, 3)   # 8
```

**注意**：默认参数放在位置参数后面

```python
# 正确
def f(a, b=1, c=2):
    pass

# 错误
def f(a=1, b):
    pass
```

## 可变参数 `*args`

```python
def sum_all(*args):
    print(type(args))   # <class 'tuple'>
    return sum(args)

sum_all(1, 2, 3)       # 6
sum_all(1, 2, 3, 4)    # 10
sum_all()              # 0
```

### 解包传参

```python
nums = [1, 2, 3]
sum_all(*nums)         # 相当于 sum_all(1, 2, 3)
```

## 关键字参数 `**kwargs`

```python
def print_info(**kwargs):
    print(type(kwargs))  # <class 'dict'>
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30)
# name: Alice
# age: 30
```

### 解包传参

```python
info = {"name": "Bob", "city": "NYC"}
print_info(**info)   # 相当于 name="Bob", city="NYC"
```

## 组合使用

```python
def func(pos1, pos2, *args, kw1=None, kw2=None, **kwargs):
    print(f"pos1={pos1}, pos2={pos2}")
    print(f"args={args}")
    print(f"kw1={kw1}, kw2={kw2}")
    print(f"kwargs={kwargs}")

func(1, 2, 3, 4, kw1="a", kw2="b", extra="c")
# pos1=1, pos2=2
# args=(3, 4)
# kw1=a, kw2=b
# kwargs={'extra': 'c'}
```

## 关键字-only 参数

强制某些参数必须用关键字形式传入：

```python
# Python 3
def greet(name, *, greeting="Hello"):
    return f"{greeting}, {name}!"

greet("Alice")           # 'Hello, Alice!'
greet("Bob", greeting="Hi")  # 'Hi, Bob!'
greet("Carol", "Hey")    # TypeError! 必须用关键字
```

### 对比位置-or-关键字

```python
# * 之前的是位置或关键字参数
def f(a, b, *, c, d):
    # a, b 可以：位置传入 f(1, 2) 或关键字 f(a=1, b=2)
    # c, d 必须：关键字 f(c=3, d=4)
```

## 参数顺序规则

牢记：**位置 → *args → 关键字-only → **kwargs**

```python
# 完整顺序
def f(pos1, pos2, *args, kw_only1, kw_only2, **kwargs):
    pass
```

## 实际应用

### 日志函数

```python
def log(level, *messages, timestamp=None, **extra):
    print(f"[{level}]", *messages)
    if timestamp:
        print(f"  @ {timestamp}")
    if extra:
        print(f"  extra: {extra}")

log("ERROR", "连接失败", "重试中", timestamp="10:00", code=500)
# [ERROR] 连接失败 重试中
#   @ 10:00
#   extra: {'code': 500}
```

### 可选参数模式

```python
def create_user(name, email, *, role="user", verified=False):
    return {
        "name": name,
        "email": email,
        "role": role,
        "verified": verified
    }

create_user("Alice", "alice@example.com")
create_user("Bob", "bob@example.com", role="admin")
```

## 小结

1. **位置参数**：必须按顺序传递
2. `*args`**：收集多余的位置参数为 tuple
3. `**kwargs`**：收集多余的关键字参数为 dict
4. **关键字-only**：`*` 后的参数必须用关键字
5. **解包**：`func(*args)` 解包 tuple/list，`func(**dict)` 解包 dict
