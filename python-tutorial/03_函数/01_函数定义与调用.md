# 函数定义与调用

> 函数是组织代码的基本单元。本章详细讲解函数的创建、调用、参数传递，以及返回值。

---

## 1. 什么是函数

**函数**：一段可以重复使用的代码，接受输入并产生输出。

```python
# 函数可以把复杂的操作封装起来
# 以下是"没有函数"的代码 - 重复且难以维护：
print("=" * 50)
print("用户管理")
print("=" * 50)
print("功能1")
print("功能2")
# ... 更多重复代码 ...
```

用函数重构后：

```python
def print_header(title):
    """打印标题栏"""
    print("=" * 50)
    print(title)
    print("=" * 50)

def print_menu():
    """打印菜单"""
    print("功能1")
    print("功能2")

# 调用函数
print_header("用户管理")
print_menu()
```

## 2. 定义函数

### 基本语法

```python
# def 函数名(参数):
#     函数体
#     return 返回值

def greet():
    """简单的问候函数"""
    print("Hello, World!")

# 调用函数
greet()  # 输出: Hello, World!
```

### 函数文档字符串

给函数添加说明，方便自己和他人理解：

```python
def add(a, b):
    """
    计算两个数的和。

    参数:
        a (int/float): 第一个数
        b (int/float): 第二个数

    返回:
        int/float: 两个数的和
    """
    return a + b

# 查看文档
print(add.__doc__)
help(add)  # 完整格式显示
```

## 3. 函数是对象

这是 Python 的重要特性：**函数是一等公民**，可以像变量一样使用：

```python
def greet():
    return "Hello!"

# 函数可以赋值给变量
f = greet          # 注意：没有括号！
print(f)            # <function greet at 0x...> ← 函数对象
print(f())          # Hello! ← 调用函数

# 函数可以作为参数
def call_twice(func):
    return func() + func()

result = call_twice(greet)
print(result)  # Hello!Hello!

# 函数可以作为返回值
def create_greeter():
    def inner():
        return "Hi!"
    return inner  # 返回函数对象

greet = create_greeter()
print(greet())  # Hi!
```

## 4. 参数传递机制

### Python 的"按对象引用传递"

很多语言有"按值传递"或"按引用传递"，Python 的机制有点不同：

```python
# 不可变对象（int, str, tuple）：像"按值传递"
def modify_int(x):
    x = 10  # 创建新的局部变量 x，不影响外面的变量

a = 5
modify_int(a)
print(a)  # 5 ← 不变！
```

图解：

```
不可变对象：
  a = 5
  │
  └──→ int(5) ←── x = 10 (创建新的 int(10))

  a 仍然指向 int(5)，不变
```

```python
# 可变对象（list, dict, set）：像"按引用传递"
def modify_list(lst):
    lst.append(4)  # 修改原对象
    lst = [100]    # 这行只影响局部变量

my_list = [1, 2, 3]
modify_list(my_list)
print(my_list)  # [1, 2, 3, 4] ← 被修改了！
```

图解：

```
可变对象：
  my_list ───→ [1, 2, 3]
                │
  lst（参数）───┘ ← 指向同一个对象

  lst.append(4) 修改的是同一个列表对象
```

## 5. 默认值参数

给参数默认值，使其变成可选：

```python
def greet(name, greeting="Hello"):
    """greeting 有默认值，可选"""
    return f"{greeting}, {name}!"

# 不提供 greeting，使用默认值
print(greet("Alice"))      # Hello, Alice!

# 提供 greeting，覆盖默认值
print(greet("Bob", "Hi"))  # Hi, Bob!
```

### 默认值参数的坑（重要！）

**默认值只计算一次！** 如果默认值是可变对象，会出问题：

```python
# 错误的写法！
def bad_append(item, target=[]):  # [] 在函数定义时创建一次
    target.append(item)
    return target

print(bad_append(1))  # [1]
print(bad_append(2))  # [1, 2] ← 预期是 [2]！
print(bad_append(3))  # [1, 2, 3] ← 越来越长！
```

图解：

```
默认值 [] 只创建一次，被所有调用共享：
  第1次调用: target 指向 []，append(1) → [1]
  第2次调用: target 仍然指向 [1]！→ [1, 2]
  第3次调用: target 仍然指向 [1, 2]！→ [1, 2, 3]
```

**正确的写法**：

```python
# 正确：用 None 作为默认值
def good_append(item, target=None):
    if target is None:      # 每次调用时检查
        target = []          # 每次创建新的空列表
    target.append(item)
    return target

print(good_append(1))  # [1]
print(good_append(2))  # [2] ← 正确！
print(good_append(3))  # [3] ← 正确！
```

## 6. return 语句

### 多个返回值

Python 函数可以返回多个值，实际上是返回一个 tuple：

```python
def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder  # 返回 tuple

result = divide(10, 3)
print(result)        # (3, 1) ← tuple
quotient, remainder = divide(10, 3)  # 解包
print(quotient)       # 3
print(remainder)      # 1
```

### 没有 return 或 return None

```python
def say_hi():
    print("hi")
    # 没有 return

result = say_hi()  # None ← 没有显式返回时，返回 None
print(result)      # None
```

### 提前返回

```python
def find_item(items, target):
    """找到就返回，找不到返回 -1"""
    for i, item in enumerate(items):
        if item == target:
            return i  # 找到就返回，函数结束
    return -1  # 循环结束都没找到

print(find_item([1, 2, 3, 4], 3))  # 2 ← 找到了
print(find_item([1, 2, 3, 4], 5))  # -1 ← 没找到
```

## 7. 变量作用域（LEGB 法则）

Python 查找变量时，按以下顺序搜索：

| 作用域 | 说明 |
|--------|------|
| **L**ocal | 当前函数内部 |
| **E**nclosed | 外层函数（嵌套函数） |
| **G**lobal | 模块级别（函数外部） |
| **B**uilt-in | Python 内置（如 print, len） |

```python
x = "global"  # Global

def outer():
    x = "enclosed"  # Enclosed

    def inner():
        x = "local"  # Local
        print(x)      # 打印 "local"

    inner()
    print(x)         # 打印 "enclosed"

outer()
print(x)             # 打印 "global"
```

### global 和 nonlocal

```python
# global：声明使用全局变量
counter = 0

def increment():
    global counter  # 声明我要用外面的 counter
    counter += 1

increment()
increment()
print(counter)  # 2 ← 修改了全局变量

# nonlocal：声明使用外层函数的变量
def outer():
    count = 0

    def inner():
        nonlocal count  # 声明我要用 outer 函数的 count
        count += 1

    inner()
    print(count)  # 1

outer()
```

## 8. 函数调用栈

```python
def a():
    print("进入 a")
    b()
    print("离开 a")

def b():
    print("进入 b")
    c()
    print("离开 b")

def c():
    print("在 c 中")

a()

# 输出顺序：
# 进入 a
# 进入 b
# 在 c 中
# 离开 b
# 离开 a
```

图解调用栈：

```
调用: a() → b() → c()

栈（后进先出）：
  c() ← 当前执行
  b()
  a()

c() 执行完毕后，栈弹出，回归 b()
```

## 9. 嵌套函数与闭包

函数内部可以定义函数：

```python
def outer():
    x = 10  # outer 的局部变量

    def inner():
        print(x)  # inner 可以访问 outer 的变量

    inner()  # 调用 inner

outer()  # 输出: 10
```

闭包：inner 函数"记住"了创建时的环境（x=10）。

## 10. 小结

### 必须掌握

1. **函数是对象**：可以赋值、传参、返回
2. **参数传递**：不可变对象不变，可变对象可能被修改
3. **默认值陷阱**：默认参数用 `None`，不用 `[]` 或 `{}`
4. **return**：可返回多值（tuple），无 return 返回 None
5. **作用域 LEGB**：Local → Enclosed → Global → Built-in
6. **global/nonlocal**：修改外层变量

### 常见错误

```python
# 错误1：在函数内部修改全局变量（不用 global）
counter = 0

def increment():
    counter = counter + 1  # UnboundLocalError！
    # 因为解释器认为 counter 是局部变量
    # 但这个局部变量还没赋值就被使用了

# 正确做法：
def increment():
    global counter
    counter = counter + 1

# 错误2：忘记 return
def get_square(x):
    x ** 2  # 忘了 return！

result = get_square(5)
print(result)  # None ← 错！

# 正确：
def get_square(x):
    return x ** 2
```
