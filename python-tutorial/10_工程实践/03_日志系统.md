# 日志系统

> print() 是用来调试的，logging 是用来记录的。本章教你搭建专业的日志系统，告别 print 调试。

---

## 1. print 调试的问题

```python
def calculate(x, y):
    print(f"DEBUG: x={x}, y={y}")        # 临时加的
    result = x / y
    print(f"DEBUG: result={result}")      # 临时加的
    return result

# 问题：
# 1. 发布时需要一个个删除
# 2. 没有时间戳、没有级别
# 3. 不能控制输出到文件/控制台
# 4. 不能按级别过滤
# 5. 输出到 stdout，混在正常输出中
```

## 2. logging 基础

```python
import logging

# 快速配置
logging.basicConfig(level=logging.INFO)

# 不同级别的日志
logging.debug("详细的调试信息")
logging.info("一般信息")
logging.warning("警告信息")
logging.error("错误信息")
logging.critical("严重错误")

# 默认只显示 WARNING 及以上
# 输出：
# WARNING:root:警告信息
# ERROR:root:错误信息
# CRITICAL:root:严重错误
```

---

## 3. 日志级别

```python
import logging

# 日志级别（从低到高）
# DEBUG < INFO < WARNING < ERROR < CRITICAL

# 设置级别
logging.basicConfig(level=logging.DEBUG)   # 显示所有
logging.basicConfig(level=logging.WARNING)  # 只显示 WARNING 及以上
logging.basicConfig(level=logging.ERROR)    # 只显示 ERROR 及以上
```

| 级别 | 值 | 使用场景 |
|------|-----|----------|
| DEBUG | 10 | 开发调试，变量值、执行路径 |
| INFO | 20 | 运行信息，请求处理、服务启停 |
| WARNING | 30 | 警告，即将废弃的功能、接近限制 |
| ERROR | 40 | 错误，功能异常但程序还能运行 |
| CRITICAL | 50 | 严重错误，程序可能无法继续 |

## 4. 日志配置详解

```python
import logging

logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s | %(levelname)-8s | %(name)s | %(message)s',
    datefmt='%Y-%m-%d %H:%M:%S',
    filename='app.log',          # 输出到文件
    filemode='a',                # 追加模式
    encoding='utf-8',
)

logging.info("服务启动")
logging.warning("磁盘使用率达到 80%")
```

### format 常用字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `%(asctime)s` | 时间 | `2024-03-15 10:30:45` |
| `%(levelname)s` | 日志级别 | `INFO` |
| `%(name)s` | logger 名称 | `root` |
| `%(filename)s` | 文件名 | `main.py` |
| `%(lineno)d` | 行号 | `42` |
| `%(funcName)s` | 函数名 | `process` |
| `%(message)s` | 日志内容 | `用户登录成功` |
| `%(threadName)s` | 线程名 | `MainThread` |

## 5. 创建 logger 实例（推荐方式）

不要用 root logger，为每个模块创建独立的 logger：

```python
import logging

# 创建 logger（惯例：用模块名）
logger = logging.getLogger(__name__)
logger.setLevel(logging.DEBUG)

# 创建 handler（控制台输出）
console_handler = logging.StreamHandler()
console_handler.setLevel(logging.INFO)

# 创建 handler（文件输出）
file_handler = logging.FileHandler("app.log", encoding="utf-8")
file_handler.setLevel(logging.DEBUG)

# 创建 formatter
formatter = logging.Formatter(
    '%(asctime)s | %(name)s | %(levelname)s | %(message)s',
    datefmt='%Y-%m-%d %H:%M:%S'
)

# 组装
console_handler.setFormatter(formatter)
file_handler.setFormatter(formatter)
logger.addHandler(console_handler)
logger.addHandler(file_handler)

# 使用
logger.debug("调试信息")      # 只写到文件
logger.info("处理请求")        # 写到文件 + 控制台
```

### 图解结构

```
Logger (logger.getLogger(__name__))
    ├── Handler 1: StreamHandler  → 输出到控制台
    │   ├── Level: INFO
    │   └── Formatter: 简单格式
    │
    └── Handler 2: FileHandler   → 输出到文件
        ├── Level: DEBUG
        └── Formatter: 详细格式
```

## 6. 配置文件方式（logging.conf）

```ini
# logging.conf
[loggers]
keys=root,myapp

[handlers]
keys=console,file

[formatters]
keys=simple,detailed

[logger_root]
level=WARNING
handlers=console

[logger_myapp]
level=DEBUG
handlers=console,file
qualname=myapp
propagate=0

[handler_console]
class=StreamHandler
level=INFO
formatter=simple
args=(sys.stdout,)

[handler_file]
class=handlers.RotatingFileHandler
level=DEBUG
formatter=detailed
args=('app.log', 'a', 10485760, 5)  # 10MB, 5个备份

[formatter_simple]
format=%(levelname)-8s %(message)s

[formatter_detailed]
format=%(asctime)s | %(name)s | %(levelname)s | %(filename)s:%(lineno)d | %(message)s
```

加载配置：

```python
import logging.config

logging.config.fileConfig("logging.conf")
logger = logging.getLogger("myapp")
logger.info("从配置文件加载的日志")
```

## 7. RotatingFileHandler 日志轮转

```python
import logging
from logging.handlers import RotatingFileHandler, TimedRotatingFileHandler

# 按文件大小轮转
handler = RotatingFileHandler(
    "app.log",
    maxBytes=10 * 1024 * 1024,  # 10MB
    backupCount=5                 # 保留5个备份
)
# 结果：app.log, app.log.1, app.log.2, ..., app.log.5

# 按时间轮转
handler = TimedRotatingFileHandler(
    "app.log",
    when="midnight",     # 每天午夜轮转
    interval=1,
    backupCount=30       # 保留30天
)
# when 可选: 'S', 'M', 'H', 'D', 'W0'-'W6', 'midnight'
```

## 8. 异常日志记录

```python
import logging
import traceback

logger = logging.getLogger(__name__)

try:
    result = 1 / 0
except Exception:
    # 方式1：记录简单信息
    logger.error("除法操作失败")

    # 方式2：记录完整异常信息
    logger.error("除法操作失败", exc_info=True)

    # 方式3：使用 exception（等价于 error(..., exc_info=True)）
    logger.exception("除法操作失败")
    # 自动包含完整的 traceback！
```

## 9. 结构化日志（JSON）

对于日志分析系统（如 ELK），输出 JSON 格式：

```python
import json
import logging

class JsonFormatter(logging.Formatter):
    def format(self, record):
        log_entry = {
            "timestamp": self.formatTime(record),
            "level": record.levelname,
            "logger": record.name,
            "message": record.getMessage(),
            "module": record.filename,
            "line": record.lineno,
        }
        if record.exc_info:
            log_entry["exception"] = self.formatException(record.exc_info)
        return json.dumps(log_entry, ensure_ascii=False)


handler = logging.StreamHandler()
handler.setFormatter(JsonFormatter())

logger = logging.getLogger("json_logger")
logger.addHandler(handler)
logger.setLevel(logging.INFO)

logger.info("用户登录", extra={"user_id": 123})
```

## 10. logging 最佳实践

```python
# 项目结构
# my_project/
# ├── my_module/
# │   ├── __init__.py
# │   ├── main.py
# │   ├── utils.py
# │   └── logger.py        ← 集中配置
# └── logging.conf          ← 配置文件
```

### logger.py（集中管理）

```python
# my_module/logger.py
import logging
import sys

def setup_logger(name: str = None) -> logging.Logger:
    """创建和配置 logger"""
    logger = logging.getLogger(name or __name__)

    if not logger.handlers:  # 避免重复添加 handler
        logger.setLevel(logging.DEBUG)

        # 控制台 handler
        console = logging.StreamHandler(sys.stdout)
        console.setLevel(logging.INFO)
        console_fmt = logging.Formatter(
            '%(levelname)-8s | %(message)s'
        )
        console.setFormatter(console_fmt)

        # 文件 handler
        file_handler = logging.FileHandler("app.log", encoding="utf-8")
        file_handler.setLevel(logging.DEBUG)
        file_fmt = logging.Formatter(
            '%(asctime)s | %(name)s | %(levelname)s | %(filename)s:%(lineno)d | %(message)s'
        )
        file_handler.setFormatter(file_fmt)

        logger.addHandler(console)
        logger.addHandler(file_handler)

    return logger
```

### 使用

```python
# main.py
from my_module.logger import setup_logger

logger = setup_logger(__name__)

def main():
    logger.info("应用启动")
    try:
        process()
    except Exception:
        logger.exception("处理失败")
    logger.info("应用结束")

# utils.py
from my_module.logger import setup_logger

logger = setup_logger(__name__)

def helper():
    logger.debug("工具函数被调用")
```

## 11. 小结

1. **print → logging**：生产环境必须用 logging
2. **每个模块一个 logger**：`logger = logging.getLogger(__name__)`
3. **配置格式**：`%(asctime)s | %(levelname)s | %(name)s | %(message)s`
4. **日志级别**：DEBUG(开发) → INFO(运行) → WARNING → ERROR → CRITICAL
5. **Handler**：控制台 + 文件（可同时）
6. **RotatingFileHandler**：防止日志文件过大
7. **exc_info=True**：记录完整异常信息
8. **不要用 root logger**：创建独立 logger，便于管理和过滤
