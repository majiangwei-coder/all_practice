## 快速上手

### 打卡工具

本目录下有配套的打卡脚本和看板：

| 文件 | 说明 |
|------|------|
| `checkin.py` | 打卡脚本 |
| `dashboard.html` | 可视化看板（用浏览器打开） |
| `data.json` | 打卡数据，脚本自动管理，不用手动编辑 |

### 打卡命令

```bash
# 进入目录
cd "d:/Program Files (x86)/practice/百日打卡计划"

# 打卡（今天）
python checkin.py

# 查看进度
python checkin.py status

# 补打卡
python checkin.py --date 2026-05-19
```

### 打卡示例

运行 `python checkin.py` 后，脚本会依次问三个目标：
- **钢琴** → 练了吗？练了多久？备注？
- **体重** → 运动了吗？什么类型？称重日？
- **英语** → 学了吗？多久？几个词？

直接输入 `y` 或 `n` 回答，打完了会自动保存到 `data.json`。

### 看板

用 VS Code 右键 `dashboard.html` → 用 Live Server 打开，或直接双击用浏览器打开（可能需要粘贴 JSON 数据）。

## 三、体重 ⚖️

### 阶段一：基线建立（Day 1-30）

**目标：** 建立运动习惯 + 清理饮食

**运动（每周 3-4 次）：**
- 快走或慢跑 20-30 分钟
- 心率控制在 Zone 1-2（能完整说话）
- 配速参考：8:00-9:00/km（可用 `跑步配速.py` 计算）

**饮食：**
- 戒含糖饮料（改喝水/茶/黑咖啡）
- 晚餐碳水减 1/4
- 每天 2L 水

**记录：** 每日晨起空腹称重，记录在打卡中

### 阶段二：系统推进（Day 31-60）

**目标：** 提高运动强度 + 优化饮食结构

**运动（每周 4-5 次）：**
- 跑步 30-40 分钟，配速 7:00-8:00/km
- 每周 2 次力量训练：
  - 俯卧撑 3×10-15
  - 深蹲 3×15-20
  - 平板支撑 3×30-60秒

**饮食：**
- 211 餐盘法：2 份蔬菜 : 1 份蛋白 : 1 份碳水
- 避免精加工零食，用水果/坚果替代

### 阶段三：巩固优化（Day 61-100）

**目标：** 突破平台期 + 养成长期习惯

**运动（每周 4-5 次）：**
- 跑步 40-50 分钟或 5km+
- 每周 1 次间歇跑：400m × 4-6 组（配速 5:30-6:00/km）
- 力量训练维持每周 2 次（可增加负重/组数）

**饮食：**
- 可选：16:8 间歇性禁食（12:00-20:00 进食窗口）
- 学会看营养成分表，关注蛋白质摄入（每公斤体重 × 1.2-1.6g）

**预期成果：** 减重 5-8%，精力提升、衣服变宽松

## 五、打卡规则

1. **每天打卡**：每个目标完成即为打卡成功
2. **补打卡**：最多补过去 3 天
3. **不可预填**：不能提前打卡未来的日期
4. **连续天数**：中断则归零（补打卡可维持连续）
5. **弹性执行**：某天确实没时间，最低完成 10 分钟也算打卡成功
