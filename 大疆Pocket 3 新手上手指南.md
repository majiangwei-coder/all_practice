## 1. 认识你的 Pocket 3

### 1.1 这是什么？

想象一下：**把一台专业摄像机缩小到口红大小，然后装在一个会自动保持平衡的"机械手臂"上**——这就是 Pocket 3。

- **它能做什么？** 拍摄超级稳定、不晃不抖的视频，比手机拍的稳得多
- **它和手机拍的区别？** 手机靠软件防抖（裁切画面），Pocket 3 靠物理云台（真正的机械稳定），画面更自然、更广、更清晰
- **难不难学？** 不难！它设计出来就是为了让你随手拍的

### 1.2 各部位名称

拿到机器后，对照着看：

```
┌─────────────────────────────────────────┐
│  ⬛ 屏幕（可翻转）                        │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │                                   │  │
│  │           触摸屏                   │  │
│  │                                   │  │
│  └───────────────────────────────────┘  │
│                                         │
│  ● 摄像头（在云台顶部）                   │
│  ● 麦克风（机身前后各一个）               │
│                                         │
│  左侧：电源键（长按开关机）                │
│  右侧：录制键（按一下开始/停止录制）        │
│  底部：Type-C 充电口 + microSD 卡槽      │
│  背面：摇杆（控制云台方向）                │
└─────────────────────────────────────────┘
```

**重要部件说明：**

| 部件 | 位置 | 作用 |
|------|------|------|
| **摄像头/云台** | 顶部可转动的部分 | 拍摄画面 + 自动保持稳定 |
| **触摸屏** | 机身正面 | 显示画面、操作菜单 |
| **电源键** | 机身左侧 | 长按开关机，短按休眠 |
| **录制键** | 机身右侧 | 红色按钮，按一次开始/停止录像 |
| **摇杆** | 机身背面 | 上下左右推动，控制镜头转向 |
| **扬声器** | 机身背部 | 播放声音 |
| **Type-C 口** | 机身底部 | 充电、连接电脑/手机 |
| **SD 卡槽** | 机身底部 | 插入存储卡 |

## 2. 开箱清单

打开包装盒，检查以下物品是否齐全：

- ✅ **Pocket 3 主机** × 1
- ✅ **保护壳** × 1（收纳时一定要套上，保护云台）
- ✅ **Type-C 数据线** × 1（充电和传数据用）
- ✅ **手绳** × 1（套在手腕上防摔落）
- ✅ **说明书/保修卡** × 1（可以先不看，看本指南就够了）
- ⬜ **microSD 存储卡** ❗**需要自己买，包装里没有！**

> ⚠️ **重要提醒：** 需要自备一张 microSD 存储卡（也叫 TF 卡），推荐购买 **SanDisk Extreme Pro 或 Samsung Pro Plus，容量 128GB 或 256GB，速度等级 U3/V30 以上**。没有卡就无法拍摄。

如果你买的是 **Creator Combo（全能套装）**，还会额外包含：
- 无线麦克风（领夹式，收音用）
- 迷你三脚架（自拍/延时摄影用）
- 广角镜（拍更广的画面）
- 续航手柄（延长使用时间）

## 3. 第一次使用前的准备

### 3.1 充电

1. 用包装里的 **Type-C 数据线** 连接机身底部的 Type-C 口
2. 另一端插到 **手机充电头**（5V/2A 或更高功率的都行）或电脑 USB 口
3. 屏幕会显示充电图标，充满约 **1.5 小时**
4. 第一次使用建议充满电

> 💡 **提示：** 充电时机器可以不开机。充电中机身微微发热是正常现象。

### 3.2 安装存储卡

1. 找到机身底部的卡槽
2. 芯片面朝屏幕方向，将 microSD 卡插入
3. 听到"咔哒"一声即到位
4. 取卡时再往里按一下，卡会弹出来

### 3.3 激活设备（非常重要！）

**第一次开机必须连接手机 App 激活才能使用。**

1. 手机下载 **DJI Mimo** App（各大应用商店搜索，或去大疆官网下载）
2. 长按机身左侧电源键开机
3. 打开手机 Wi-Fi 和蓝牙
4. 打开 DJI Mimo App
5. App 会自动搜索到 Pocket 3，按提示连接
6. 按照 App 引导完成激活
7. 如果有固件更新提示，建议连接 Wi-Fi 更新（会优化性能和修复 bug）

> ⚠️ **不激活的话，机器只能看，不能拍！**

## 4. 开关机与基础按键操作

### 4.1 开关机

| 操作 | 怎么做 |
|------|--------|
| **开机** | 长按左侧电源键约 **2 秒**，屏幕亮起 |
| **关机** | 长按左侧电源键约 **2 秒**，屏幕关闭 |
| **休眠/唤醒** | 短按一下电源键，快速息屏省电；再短按唤醒 |

### 4.2 录制键（右侧红色按钮）

- **按一下**：开始录像 / 停止录像
- **拍照模式下**：按一下拍照

### 4.3 摇杆（机身背面）

| 操作 | 效果 |
|------|------|
| 往上推 | 镜头向上转 |
| 往下推 | 镜头向下转 |
| 往左推 | 镜头向左转 |
| 往右推 | 镜头向右转 |

> 💡 **新手提示：** 推得越用力，镜头转得越快。轻轻推可以拍出平滑的运镜效果。

### 4.4 屏幕翻转

- 屏幕可以向 **左翻转 90°**，变成横屏拍摄模式
- 这是 Pocket 3 的一大特色——**横竖屏切换超方便**
- 自拍时把屏幕翻转 180° 朝向自己

## 5. 屏幕操作详解

### 5.1 主界面布局

开机后屏幕上你会看到：

```
┌──────────────────────────────────┐
│ 🔙  ←  [模式]  [参数]  [电量]    │  ← 顶部状态栏
│                                  │
│                                  │
│          取景画面（实时预览）      │
│                                  │
│                                  │
│   [0.5x] [1x] [2x]   ⬆           │  ← 变焦 + 曝光调节
│                                  │
│   ⬜  ●  ⏺  📷  ⏸              │  ← 底部工具栏
│   (回放) (录像) (拍照) (慢动作)    │
└──────────────────────────────────┘
```

### 5.2 基本手势操作

| 手势 | 作用 |
|------|------|
| **从右向左滑** | 进入**设置菜单**（调参数的地方） |
| **从左向右滑** | 查看**已拍的照片/视频**（回放） |
| **从下往上滑** | 展开**详细参数面板**（分辨率、帧率等） |
| **点按** | 选择/确认 |
| **双指捏合** | 放大/缩小预览画面（不是变焦） |

### 5.3 底部拍摄模式切换

屏幕底部一排图标，从左到右：

| 图标 | 模式 | 什么时候用 |
|------|------|-----------|
| ⬜（回放） | 查看已拍内容 | 拍完后检查效果 |
| ●（录像） | 拍视频（最常用） | 大多数时候就用这个 |
| 📷（拍照） | 拍照片 | 需要拍照时 |
| 🕐（慢动作） | 慢动作视频 | 拍流水、运动等慢镜头 |
| ⏸（延时摄影） | 延时摄影 | 拍日出日落、车流等 |

### 5.4 快速参数调节

在录像模式下，点屏幕上的数字：

- **变焦：** 屏幕右侧有 0.5x / 1x / 2x 等数字，点击切换广角/标准/放大
- **曝光补偿（EV）：** 变焦数字旁边的 `⬆` 或数值，上下滑动调节画面明暗

## 6. 连接手机（DJI Mimo App）

### 6.1 为什么要连手机？

- ✅ 用手机屏幕当**大监视器**（看得更清楚）
- ✅ **遥控拍摄**（手机上操作，机器架远处）
- ✅ 一键**导出素材**到手机相册
- ✅ 固件升级
- ✅ 用手机剪视频

### 6.2 无线连接

1. 打开 Pocket 3 和手机蓝牙
2. 打开 DJI Mimo App
3. App 会自动发现 Pocket 3，点击连接
4. 连接成功后，手机屏幕会显示相机画面

> 💡 首次连接后，之后打开 App 会自动重连。

### 6.3 有线连接（推荐，更稳定）

1. 用 Type-C 数据线连接 Pocket 3 和手机
2. 打开 DJI Mimo App
3. 自动进入有线连接模式
4. **优点：** 零延迟、不耗 Wi-Fi、传输素材更快

## 7. 第一次拍摄：一步一步来

跟着做，拍出你的第一条视频：

### 第一步：准备工作
1. 确保电量充足
2. 确保存储卡已插入
3. 取下保护壳

### 第二步：开机
1. 长按电源键 2 秒开机
2. 云台会自动回正（镜头朝前）

### 第三步：选择模式
1. 确认屏幕底部选中的是 **●（录像）** 模式
2. 如果不是，点一下 ● 图标

### 第四步：构图
1. 手持机身，对准你要拍的画面
2. 用摇杆调整镜头方向
3. 想看屏幕更清楚？翻转屏幕到适合角度

### 第五步：开始录制
1. 按一下右侧的 **红色录制键**
2. 屏幕上会出现 **● REC** 字样和计时
3. 开始移动拍摄吧！**正常走路就行，画面会非常稳**

> 💡 **新手必知：** 不用像举手机那样刻意端稳，Pocket 3 的云台会自动帮你稳住画面。你只需关心拍什么，稳的事情交给它。

### 第六步：停止录制
1. 再按一下右侧的红色录制键
2. 录制结束，视频自动保存

### 第七步：回放检查
1. 从屏幕左向右滑动
2. 可以看到刚刚拍的视频
3. 点播放按钮查看效果

### 第八步：关机收纳
1. 长按电源键关机
2. 云台会自动收回
3. 套上保护壳再放包里

## 8. 核心拍摄模式详解

### 8.1 录像模式（●）⭐ 最常用

拍摄普通视频。进入后可以设置的参数：

| 参数 | 推荐设置（新手） | 说明 |
|------|-----------------|------|
| **分辨率** | 4K | 4K 最清晰，也有 1080p 可选 |
| **帧率** | 30fps | 30 流畅，60 更流畅但也更占空间 |
| **编码** | H.264 | 兼容性最好；H.265 文件更小但老设备播不了 |
| **色彩** | 普通 | D-Log M 是给后期调色用的，新手先别碰 |

**简单理解：**
- 4K = 超高清，1920×1080 的四倍清晰度
- 30fps = 每秒 30 张画面，看起来流畅
- 60fps = 每秒 60 张画面，更流畅，也适合后期做慢动作

### 8.2 拍照模式（📷）

拍照片用。支持：
- **单拍**：按一下就拍一张
- **全景**：转动云台自动拍多张合成广角全景照片

### 8.3 慢动作模式（🕐）

拍出慢镜头效果：
- 通常用 **4K 120fps** 或 **1080p 240fps**
- 拍完后回放自动变慢
- 适合拍：跳水、奔跑、宠物甩头、风吹头发

### 8.4 延时摄影（⏸）

把几十分钟压缩成几秒：
- 适合拍：日出日落、云彩流动、车水马龙、人来人往
- **必须用三脚架！** 放在一个位置不动，拍 15-30 分钟
- 机器会自动合成视频

## 9. 云台模式：它为什么这么稳

### 9.1 什么是云台？

传统理解：云台就是那个让镜头"悬浮"起来的机械结构。**它有三轴，分别管上下、左右、旋转**，三个轴配合工作，你手再怎么抖，镜头都能保持水平。

### 9.2 云台模式切换

从屏幕 **从右向左滑** → 找到 **云台模式**，有三种：

| 模式 | 图标 | 效果 | 什么时候用 |
|------|------|------|-----------|
| **增稳**（默认） | 一个箭头 | 镜头始终朝一个方向，你走路上下颠簸也不影响画面 | **日常手持拍摄，最常用** |
| **俯仰锁定** | 弧线箭头 | 上下方向锁定，只能左右转 | 拍平移镜头（比如从左到右扫过风景） |
| **FPV** | 箭头转弯 | 完全跟随手部动作，像第一人称视角 | 拍快速运动、创意运镜 |

> ⚠️ **新手建议：** 先一直用 **增稳模式**（默认），拍熟了再尝试其他模式。

### 9.3 智能跟随（ActiveTrack）

**非常实用的功能！**

1. 在屏幕上**用手指框选**一个人或物体
2. 云台会自动跟着这个目标转动
3. 你只需要拿着机器走，目标会始终保持在画面中央

> 💡 **适用场景：** 拍走动的人、跑动的宠物、骑行的朋友等。

## 11. 如何把视频/照片传到手机或电脑

### 方法一：无线传到手机（最方便）

1. 打开 DJI Mimo App 并连接 Pocket 3
2. 在 App 中进入 **「相册」**
3. 选择要下载的素材
4. 点击下载，视频会保存到手机相册
5. **传输速度：** 约 10-20 MB/s，一分钟 4K 视频大约传 1-2 分钟

### 方法二：有线传到电脑（最快）

1. 用 Type-C 数据线连接 Pocket 3 和电脑
2. Pocket 3 屏幕上会弹出一个选项，选择 **「文件传输」**
3. 电脑上会多出一个盘符（像 U 盘一样）
4. 打开 DCIM 文件夹，里面的视频文件直接复制到电脑

> 💡 **视频文件在哪？** SD 卡根目录下的 `DCIM/` 文件夹内，视频为 `.mp4` 格式，照片为 `.jpg` 或 `.dng` 格式。

### 方法三：取出 SD 卡用读卡器

1. 关机
2. 按压卡槽中的 SD 卡，弹出
3. 插入读卡器连电脑
4. 这是**最通用的方法**，速度也最快

## 12. 常见问题

### ❓ 开机后云台不动/歪斜？

- 这是正常的！开机时云台会自检，短暂晃动一下然后回正
- 如果长时间歪斜，把机器放平在桌面上，进入设置 → 云台 → 自动校准

### ❓ 拍摄时画面突然变暗？

- 可能是切换到了暗光场景，或者手指遮挡了镜头
- 检查镜头是否清洁，手指有没有挡住

### ❓ 为什么按录制键没反应？

- 检查 SD 卡是否已插入
- 检查 SD 卡是否已满
- 检查电量是否充足
- 检查是否已完成激活

### ❓ 视频拍出来很晃？

- 检查云台模式是否选择了"增稳"模式
- 检查云台是否被卡住（保护壳有没有取下来？）

### ❓ 电池能用多久？

- 连续录制约 **2 小时**（4K 30fps）
- 实际使用中（开开停停）约可支撑一天的外出拍摄
- 建议出门带充电宝，边充边拍

### ❓ 支持边充电边拍吗？

- 支持！用充电宝可以边充边拍，长时间录制没问题

### ❓ 能防水吗？

- **不能！** Pocket 3 不防水，下雨天要防水，不能下水拍摄

### ❓ 可以当电脑摄像头吗？

- 可以！用 Type-C 连电脑，选择 **「摄像头模式」**，电脑会识别为一个摄像头
- 适合视频会议、直播

---

## 13. 保养与维护

### ✅ 每次用完后的习惯

1. **关机后再收纳**——云台会自动回到保护位置
2. **套上保护壳**——云台很精密，包里一压就可能损坏
3. **擦拭镜头**——用柔软的布（眼镜布就可以）轻轻擦拭镜头

### ✅ 每周/每月检查

- 检查镜头表面是否有灰尘或指纹
- 检查云台转动是否顺滑
- 检查 Type-C 口和卡槽是否干净

### ⚠️ 千万别做的事

| ❌ 不要 | 为什么 |
|---------|--------|
| 开机状态下硬掰云台 | 会损坏电机 |
| 不套保护壳直接扔包里 | 云台容易碰坏 |
| 用纸巾/衣服擦镜头 | 会刮花镜片 |
| 进水/淋雨 | 不防水，进水基本报废 |
| 放在高温车里暴晒 | 电池会鼓包，有危险 |
| 长期不用不充电 | 电池会饿死，建议每 3 个月充一次 |

## 14. 术语表（看不懂的来这里查）

| 术语 | 简单解释 |
|------|---------|
| **4K** | 超高清分辨率，3840×2160，约 800 万像素 |
| **1080p** | 全高清分辨率，1920×1080 |
| **fps** | 每秒帧数，数值越高画面越流畅 |
| **H.264 / H.265** | 视频压缩格式，H.265 文件更小但需要新设备才能播放 |
| **HEVC** | 就是 H.265 的另一种叫法 |
| **D-Log M** | 一种"灰片"色彩模式，保留更多细节，需要后期调色 |
| **EV（曝光补偿）** | 调亮/调暗画面，负值变暗，正值变亮 |
| **ISO** | 感光度，越高画面越亮但噪点也越多 |
| **快门速度** | 每帧曝光的时间，1/50s 表示每帧 1/50 秒 |
| **云台** | 让镜头保持稳定的机械装置 |
| **三轴** | 云台的三个旋转轴：俯仰（上下）、航向（左右）、横滚（旋转） |
| **microSD / TF 卡** | 小尺寸存储卡，Pocket 3 用它存视频和照片 |
| **U3 / V30** | SD 卡的速度等级，U3 意味着最低写入速度 30MB/s |
| **固件** | 机器内部的系统软件，类似手机的系统更新 |
| **RTMP** | 直播推流协议，可以用 Pocket 3 做直播 |
| **变焦** | 放大画面，分光学变焦（画质不变）和数码变焦（放大后变模糊） |
| **广角** | 镜头可拍到的范围，Pocket 3 等效 20mm 广角 |

## 最后的建议：新手学习路线图

```
第1天：📦 开箱 → 充电 → 插卡 → 激活 → 随便拍两段
第2天：🎬 反复练习"开机→拍摄→回放→关机"流程
第3天：📱 学会连接 App 导出视频到手机
第1周：🎥 尝试不同场景拍摄（室内/室外/白天/夜晚）
第2周：🔄 尝试不同云台模式（增稳→俯仰锁定→FPV）
第1个月：🎨 开始尝试手动调参数（色彩、曝光等）
第2个月+：✂️ 学习用手机/电脑剪辑视频

不急，慢慢来，每次学会一个功能就好。
```
